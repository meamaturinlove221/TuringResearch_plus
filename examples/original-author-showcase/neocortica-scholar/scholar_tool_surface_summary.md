## Short Summary

Neocortica-Scholar demonstrates a paper-oriented MCP tool surface for academic work. The workflow separates paper search, paper content retrieval, reference extraction, and structured paper reading. It also emphasizes public configuration through MCP env blocks, fake/live boundaries, and clear tool documentation.

## What This Demonstrates in TuringResearch

TuringResearch uses this showcase to explain its scholar pipeline:

- three-pass reading templates;

- optional live adapters with fake mode available by default;
- public-safe MCP configuration examples.
