# TuringResearch VGGT Case Study

Status: split-repo skeleton / not a real repository.

This skeleton demonstrates what a future `turingresearch-vggt-case` repository
could contain.

The flagship TuringResearch Plus repository remains the install, docs, release,
and star entry point.

## What This Is

This is a public-safe case-study skeleton showing how TuringResearch Plus was
dogfooded on a VGGT human-prior workflow.

It demonstrates:

- evidence management;
- route changes;
- failure and blocker tracking;
- advisor pack structure;
- redaction and claim safety;
- what remains human research.

## What This Is Not

- Not a VGGT experiment source repository.
- Not a training or inference repository.
- Not a raw data repository.
- Not a SMPL-X model bundle.
- Not proof of final research success.
- Not a SparseConv3D success claim.

## Contents

- `CASE_STUDY.md`: public-safe case-study draft.
- `PRIVACY.md`: public safety and privacy boundaries.
- `manifest.yaml`: skeleton file inventory and claim boundaries.

## Safety Boundary

This skeleton contains only public-safe case-study text and fake/demo artifact
references. It contains no private paths, no raw data, no SMPL-X files, no API
keys, and no unsupported experiment claims.

Human review is required before any real split repository is created.
