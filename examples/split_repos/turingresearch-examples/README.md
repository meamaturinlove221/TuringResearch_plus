# TuringResearch Examples

Status: split-repo skeleton / not a real repository.

This skeleton shows what a future `turingresearch-examples` repository could
look like.

The flagship TuringResearch repository remains the install, docs, release,
and star entry point.

## Purpose

This future examples repo would provide public-safe walkthrough material for
TuringResearch:

- public demo suite;
- project template examples;
- demo workspace;
- dashboard demo;
- paper demo;
- advisor pack demo.

## Safety Boundary

All examples must be fake/demo unless explicitly reviewed otherwise.

This skeleton does not contain:

- private VGGT files;
- raw data;
- model files;
- API keys or tokens;
- huge artifacts;
- real private logs;
- private advisor feedback;
- unsupported experiment success claims.

## What This Repo Would Demonstrate

- evidence ledger;
- artifact audit;
- route DSL;
- paper scaffold;
- dashboard;
- advisor pack;
- privacy-first public demo policy.

## What This Repo Would Not Claim

- no automatic research completion;
- no final paper generation;
- no VGGT experiment success;
- no demo result promoted to observed evidence.

## Current Status

Design-only. Do not publish or extract this skeleton until the public demo and
privacy gates pass on the exact export set.
