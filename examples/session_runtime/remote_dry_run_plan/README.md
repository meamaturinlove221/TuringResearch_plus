# Optional Remote Dry-run Plan Demo

Status: fake/demo only.

This demo shows how TuringResearch can prepare a review-only remote dry-run
plan for a human-operated pod workflow. It does not SSH, SFTP, use tmux, call
Modal, run remote commands, upload data, or write the Evidence Ledger.

Files:
- `REMOTE_DRY_RUN_PLAN.md`: generated review plan.

The remote target is a placeholder. Any real target, credential, or command
must be supplied outside this public demo and reviewed by a human.
