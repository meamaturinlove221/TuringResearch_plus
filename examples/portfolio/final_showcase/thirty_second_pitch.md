# Thirty-second Pitch

TuringResearch Plus is a local-first Research OS for keeping research project
state honest. It connects evidence ledgers, artifact audits, experiment route
specs, paper review, dashboards, advisor packs, plugin manifests, and privacy
gates.

The key rule is simple: planned work is not observed evidence. Fake/demo flows
run locally by default, live adapters are optional, and human review remains
required before any research claim is treated as real.
