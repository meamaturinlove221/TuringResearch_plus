# Scholar Demo

This folder contains a fake/default Scholar tool walkthrough.

It does not require an API key. It does not download papers. It does not bypass
paywalls. It does not mark fake citations as verified.

Files:

- `fake_paper_search.json`: deterministic fake paper search output.
- `fake_paper_content.md`: cached Markdown paper content for local reading.
- `fake_reference_report.md`: fake/cached reference report for human review.

Live mode is opt-in only and belongs in private local configuration. Public
demo files remain fake/default.

Safety boundaries:

- fake mode default;
- no real API key required;
- no paper download;
- no paywall bypass;
- no verified citation claim;
- human review required.
