# VGGT-like Public Demo Project

Status: demo only.

This project is a public-safe VGGT-like research workflow demo. It uses fake
metadata to show how a project can organize intent, evidence, artifacts,
related work, advisor notes, and a static dashboard.

It is not a VGGT experiment, not a private project mirror, and not evidence of
model success. All entries require human review before any research claim.

## Files

- `north_star.md`
- `evidence_ledger.json`
- `artifact_index.md`
- `related_work.md`
- `advisor_pack.md`
- `dashboard.html`

## Safety Boundary

- Demo only.
- Fake data only.
- No private project material.
- No API key or token.
- Planned work is not treated as a result.
