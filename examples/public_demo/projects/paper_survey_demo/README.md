# Paper Survey Public Demo Project

Status: demo only.

This project demonstrates a paper-survey workflow with fake paper notes,
review requirements, and advisor-facing synthesis placeholders.

It does not contain real citation claims, final related-work prose, or private
notes. Human review is required before using any paper statement.

- `north_star.md`
- `evidence_ledger.json`
- `artifact_index.md`
- `related_work.md`
- `advisor_pack.md`
