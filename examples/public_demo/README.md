# TuringResearch Public Demo Suite

Status: demo only.

This directory contains fake, public-safe examples that demonstrate
TuringResearch Plus workflows without exposing private data.

## Demo Coverage

- Evidence ledger
- Artifact audit
- Visual audit
- Paper method card
- Related work positioning
- Route DSL
- Failure taxonomy
- Advisor pack
- Static dashboard

## Safety Boundary

- All data is fake/demo.
- No private project path is included.
- No private model file is included.
- No data payload is included.
- No API key or token is included.
- Demo material is not experiment evidence.
- Planned demo steps are not observed results.
