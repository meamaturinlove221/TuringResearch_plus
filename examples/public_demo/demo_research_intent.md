# Demo Research Intent

Status: demo only.

## Topic

Evidence-led research workflow for a fictional geometry-aware model adapter.

## North Star

Show how TuringResearch organizes research intent, evidence, artifacts,
visual readiness, method notes, routes, failures, advisor outputs, and a static
dashboard without using private data.

## Demo Boundary

- This is not a real experiment.
- This is not a real paper review.
- This is not a production claim.
- Human review would be required before applying this pattern to a real project.
