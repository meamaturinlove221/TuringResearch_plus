# Round 205 - v1.1 Scope Lock

Status: complete.

## Goal

Lock v1.1 scope around post-v1 stabilization and the highest public-value
surfaces. This round does not implement features, create repositories, publish,
tag, or push.

## Output

- `docs/v1.1.0-final-scope.md`
- `docs/v1.1.0-implementation-order.md`
- `docs/v1.1.0-test-plan.md`
- `docs/v1.1.0-final-risk-register.md`
- `docs/v1.1.0-non-goals-final.md`
- `race/priority_board.md`

## Locked Scope

1. v1.0 baseline stabilization.
2. `turingresearch-vggt-case` split-ready finalization.
3. `turingresearch-examples` split-ready finalization.
4. Docs site.
5. Local server dashboard.
6. Paper writing beta.
7. More public demo cases.
8. Case study gallery.
9. CI/CD release hardening.
10. v1.1 release candidate.

## Non-goals

- no SaaS;
- no cloud user system;
- no unknown plugin execution;
- no default networking;
- no automatic private data upload;
- no automatic real VGGT/Modal experiment execution;
- no OS-level plugin sandbox;
- no large frontend application;
- no automatic final paper generation;
- no guaranteed star growth.

## First Priority

v1.1 starts with baseline stabilization because Round 204 confirmed the v1.0
archive is a planning baseline, not a verified public release baseline.
