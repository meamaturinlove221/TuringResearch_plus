## Deferred Candidates

- NAS / SMB Shared Artifact Store
- SSH / SFTP Remote Artifact Reader
- GitHub Artifact Sync
- Modal Run Dashboard
- Advisor Pack richer export with PPTX/PDF

## Rationale

v0.3 Sprint 1 already solved the local planning side: context handoff, pod pack,
and cache-first scholar workflow. Sprint 2 should now close the public source
and reasoning loop:

- fetch public project pages and README files safely;
- turn method cards and citation graphs into positioning notes;
- make skill routing explicit;
- improve vault graph structure;
- add ontology SOPs without copying upstream code.

## Non-Goals

- No default networking.
- No remote execution.
- No NAS / SMB / SSH / GitHub artifact sync.
- No Modal dashboard.
- No PPTX/PDF advisor export.
- No final paper conclusions.
