# v0.5.0 Beta Non-Goals

Status: planning.

Round: 100.

## Non-Goals

- No full SaaS UI.
- No user account system.
- No cloud deployment.
- No automatic paper writing.
- No automatic experiment execution.
- No default live remote sync.
- No default live web or scholar calls.
- No automatic Modal execution.
- No upload of private data.
- No raw data or private model file packaging.
- No automatic promotion of fake/demo results to observed evidence.
- No public plugin marketplace release unless explicitly scoped later.

## Release Boundary

v0.5 beta should make alpha capabilities more reliable and presentable. It
should not introduce broad new runtime obligations or weaken evidence
boundaries.
