# TuringResearch Plus v0.2.0 Sprint 2 Feature Capsules

Round 45 turns the locked Sprint 2 scope into five Feature Capsule skeletons.
This is design scaffolding only. It does not implement business logic, run VGGT,
read private VGGT paths, call network services, or claim experiment success.

round accepts them into the public MCP surface. New `paper.*` proposals must
also be aligned with `docs/mcp-tools.md` before becoming public tools.

## Sprint 1 Reuse

- Evidence Ledger supplies status boundaries and EvidenceRefs.
- Artifact Auditor supplies bundle, board, and safety context.
- Visual Evidence Auditor supplies readiness and missing visual proof.
- Advisor Pack Builder consumes route, gate, and failure summaries.
- PDF Phase B and Figure Registry supply figure/table provenance for method
  cards and architecture drafts.

## Non-Goals

- No VGGT execution.
- No Modal live execution.
- No network.
- No private `D:/vggt` reads.
- No heavy OCR.
- No full automatic paper writing.
- No fabricated VGGT result.
- No reintroduction of old project names.
