# v1.3.0 Non-goals

Status: scope lock non-goals.

Round: 259.

v1.3 focuses on full original parity. The following are not part of this scope:

- No ARIS implementation.
- No cross-model review implementation.
- No meta-optimize implementation.
- No proof-checker implementation.
- No paper-claim-audit implementation.
- No default SSH execution.
- No default networking.
- No automatic remote command execution.
- No automatic experiment execution.
- No automatic evidence ledger mutation from remote or model output.
- No automatic child repository creation.
- No full SaaS or cloud user system.
- No final paper automation.
- No breakage of the current runnable mainline.

## Allowed Study Work

ARIS study docs may remain as planning material, but study docs do not authorize
runtime implementation.

## Release Rule

If a change requires live credentials, remote execution, automatic claim
promotion, or final-paper automation, it is outside v1.3 full original parity.
