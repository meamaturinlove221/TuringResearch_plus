# v0.7.0 Non-goals

Status: scope locked.

v0.7 does not include:

- SaaS.
- Cloud-hosted user system.
- Unknown third-party plugin execution.
- License bypass.
- Automatic private data upload.
- Paper conclusions that have not been confirmed by humans.
- Default live networking.
- Automatic VGGT or Modal experiment execution.
- Final paper writing.
- Legal compliance guarantees.
- Online plugin marketplace publishing.
- Automatic installation of community plugins.
- Promotion of fake/demo/planned results to observed evidence.
- Reintroduction of prior project naming.

## Explicit Boundaries

- Trusted plugin loading means reviewed local manifests only.
- Sandbox policy is required before runtime plugin expansion.
- Dataset/license assistant produces review reports, not legal decisions.
- VGGT case study is a dogfooding case until real evidence supports stronger
  claims.
