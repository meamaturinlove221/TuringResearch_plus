## Human Review Required

Before any real split:

- maintainer approves repository creation;
- exact export trees are frozen;
- privacy and compliance scans are rerun on the exact export trees;
- claim-safety review is repeated;
- README links are reviewed;
- main repo quickstart and install path are verified unchanged.
