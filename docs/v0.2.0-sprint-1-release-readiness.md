# TuringResearch Plus v0.2.0 Sprint 1 Release Readiness

## Decision

Status: ready for maintainer review with known limitations.

Sprint 1 is suitable as a local, fake/dry-run, evidence-infrastructure slice.
It is not a live VGGT experiment release and not a public claim of paper or
advisor success.

## Ready

- VGGT/SMPL-X Evidence Ledger minimal implementation.
- Artifact Auditor minimal implementation.
- Visual evidence dry-run outputs and readiness gate.
- Advisor Pack Builder Markdown output.
- PDF Phase B lightweight figure/table extraction.
- Paper figure registry integration for PDF assets.
- Contract and workflow tests for the integrated fake path.

## Not Ready

- Full Visual Evidence Auditor image classification implementation.
- Real VGGT experiment execution.
- Real `local_scan_visual_inventory.md` evidence promotion.
- SparseConv3D success claim.
- OCR, MinerU, Marker, or complex PDF layout parsing.
- Future Sync Adapters.

## Required Before External Release

- Human review of advisor pack wording.
- Human review of source hygiene and private-path boundaries.
- Optional fixture expansion for visual evidence classification.
- Maintainer decision on whether capsule-local `vggt.*`, `artifact.*`,
  `visual.*`, and `advisor.*` should become public MCP namespaces.

## Gate Result

The Sprint 1 fake/dry-run integration gate passed local workflow, contract,
unit, name integrity, and type-check validation. Release readiness remains
scoped to local evidence infrastructure, not live VGGT results.
