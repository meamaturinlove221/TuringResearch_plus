# v1.5.0 Release Notes

Status: release-candidate notes.

Round: 357.

TuringResearch Plus v1.5.0 is a public externalization release-candidate line.
It builds on v1.4 original repo production parity by preparing local docs
deployment artifacts, manual split-repo packs, optional live polish, and a
static dashboard showcase.

This release does not publish a public site, create child repositories, run
live providers, enable live networking by default, publish PyPI artifacts, or
implement ARIS.

## Highlights

- Docs deployment dry-run.
- Docs navigation polish.
- Split repo manual packs.
- Optional live Scholar/Web/SFTP polish.
- Live safety gate.
- Dashboard landing page.
- Parity showcase view.
- Interview demo view.

## What Is New Since v1.4

- Public docs deployment is prepared as a dry-run package with manifest hashes,
  build hardening, and no real public URL.
- Docs navigation is polished around the external visitor path.
- Split repository work is packaged as manual packs for human review, without
  automatic repository creation or external push.
- Optional live Scholar, Web/Apify, and SFTP docs now have explicit skipped
  tests, env policy checks, and a unified live safety gate.
- Dashboard showcase now includes a landing page, parity view, and interview
  demo view for local/static presentation.
- v1.5 full replay and security/privacy gates cover the new public
  externalization surfaces.

## What Remains Review-gated

- Public docs deployment.
- Child repository creation.
- External push.
- GitHub release publication.
- PyPI publication.
- Live provider use.
- Live SSH/SFTP use.
- ARIS study or implementation.

## Release Boundary

v1.5.0 remains a local-first release candidate. Human review is required before
publication, claims, external repository creation, live provider usage, live
SSH/SFTP usage, or release actions.
