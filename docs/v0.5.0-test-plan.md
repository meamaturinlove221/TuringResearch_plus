# v0.5.0 Test Plan

Status: scope locked.

## Required Test Classes

- Unit tests for every new model.
- Markdown/JSON export tests.
- Fake/default workflow tests.
- Contract schema tests.
- Name integrity tests.
- Package import tests.
- CLI smoke tests where relevant.
- Public demo suite regression tests.
- No secret/raw-data/SMPL-X model packaging tests.

## Feature-Specific Tests

- Lightweight Dashboard UI: renders fixture reports; no live server required for
  default tests unless explicitly scoped.
- Advisor PDF/PPTX Export: preserves limitations and does not fabricate charts.
- Project Template Generator: creates clean generic project skeletons.
- Packaging Polish: CLI/MCP registry stays consistent.
- Public Demo Suite: runs offline from fixtures.
- VGGT Dogfooding Replay: uses exported artifacts only and writes proposed
  updates, not evidence promotions.

## Live / Manual Tests

Live tests remain skipped by default. Any UI preview or binary export smoke test
must be explicit and must not require secrets.
