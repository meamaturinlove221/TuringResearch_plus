# TuringResearch Plus v0.4.0 Release Notes

Status: release preparation complete.

Release date: 2026-05-21.

v0.4.0 extends TuringResearch Plus from local review workflows into a
remote-artifact-aware, dashboard-ready, paper-digest-capable review system. It
keeps the same evidence discipline: remote artifacts are not verified evidence,
dashboards do not execute experiments, and Markdown exports do not become
binary advisor deliverables.

## Added

- GitHub Artifact Sync for fake/default GitHub artifact metadata and selected
  small review files.
- SSH/SFTP Remote Reader for read-only remote artifact metadata and small
  reports.
- NAS/SMB Shared Store scanner for already mounted local paths.
- Cloud Object Artifact Index for provider-neutral S3/R2/OSS/GCS-style object
  manifests.
- Remote Artifact Integration for unified `ArtifactRef` and
  `UnifiedRemoteArtifactReport` outputs.
- Modal Run Dashboard for Markdown-first rendering of existing Run Ingestor
  reports.
- Experiment Board Index and Run Comparison for metadata-level cross-run
  comparison.
- Paper Digest / Three-pass Reading expansion for review-required paper digests
  and method-card bridge outputs.
- Advisor Markdown Export Bundle for future PDF/PPTX/DOCX/HTML export sources.
- v0.4 Integration Gate for fake/default end-to-end review workflows.

## Changed

- Package version advanced to `0.4.0`.
- v0.4 docs now group remote artifact, dashboard, run comparison, paper digest,
  and advisor export boundaries together.
- Release posture remains review-first: retrieved artifacts and web/paper notes
  are not automatically verified evidence.

## Fixed

- Added v0.4 contract checks for default-off/live-optional behavior and review
  boundaries.
- Added v0.4 fake end-to-end workflow tests covering remote artifacts, dashboard
  comparison, paper digest, related work/collision, and advisor Markdown bundle.
- Added dashboard and comparison checks that keep SparseConv3D success unclaimed
  without real backend evidence.

## Known Limitations

- Live remote sync is disabled by default.
- No automatic remote command execution is supported.
- TuringResearch does not execute Modal.
- PPTX/PDF export is not implemented yet.
- No UI dashboard is included yet.
- Automatic paper conclusions are not generated.
- No paywall bypass is supported.
- Raw data and SMPL-X model files are not packaged.

## Release Boundary

v0.4.0 is suitable for fake/default review workflows and maintainer review. It
must not be presented as live remote sync validation, real Modal experiment
execution, full paper reading, or binary advisor export.
