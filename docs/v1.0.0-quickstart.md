# v1.0.0 Public Quickstart

Status: public-safe quickstart draft.

Round: 179.

This path is designed to take 5-10 minutes on a local machine. It does not
require API keys, VGGT data, SMPL-X files, network access, live adapters, or
private research data.

## 0. Boundary

Everything in this quickstart is demo only.

- No real experiment is executed.
- No final paper conclusion is generated.
- No live adapter is enabled.
- No unknown plugin is executed.
- Demo evidence is not observed research evidence.

## 1. Install

From the repository root:

```powershell
python -m pip install -e .[dev]
```

Optional MCP smoke path:

```powershell
python -m pip install -e .[dev,mcp]
```

## 2. Run The Public Demo Checks

```powershell
python -m pytest tests/workflow/test_public_demo_suite.py tests/workflow/test_public_demo_expansion.py -q
```

Expected result:

```text
13 passed
```

The exact count may change as demo tests grow, but the workflow should pass
without network access or credentials.

- stdio manifest or health JSON;

- no plugin tools enabled by default.

## 8. Live Adapter Boundary

Live adapters are disabled by default.

Default config values:

```text
TURINGRESEARCH_MODE=fake
TURINGRESEARCH_ENABLE_LIVE_TESTS=0
TURINGRESEARCH_ENABLE_PLUGINS=0
TURINGRESEARCH_ENABLE_PLUGIN_LIVE_MODE=0
```

Live mode requires explicit opt-in in a private local config. Do not commit
real credentials.
