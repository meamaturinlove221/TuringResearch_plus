# v0.7.0 Test Plan

Status: scope locked.

## Baseline Checks

- Name integrity tests.
- Skills integrity tests.
- Feature capsule completeness checks.
- Package import and CLI/MCP metadata checks.
- Privacy gate and public release hygiene checks.
- Quality / regression gate.

## Feature-specific Test Tracks

| Feature | Test focus |
| --- | --- |
| Trusted Local Plugin Loading | trusted-only manifest allowlist, no unknown entrypoint execution, disabled third-party defaults |
| Plugin Sandbox Policy | permission matrix, forbidden execute/network/secret paths, policy report |
| Plugin Compatibility Harness | fake plugin fixtures, schema compatibility, no core tool override |
| MCP Distribution Polish | install docs, smoke commands, config examples, no-key defaults |
| Dashboard Refinement | static render quality, required sections, no experiment-result claims |
| Real Advisor PDF Export | optional adapter plan, source manifest, no external converter by default |
| Real Advisor PPTX Export | optional adapter plan, slide mapping, no generated charts without evidence |
| Dataset / License Compliance | policy findings, human-review flags, no legal conclusion claims |
| Local-first Research Vault UI | static/local view tests, no server required, no private data upload |
| Paper Deep Review Mode | source status, citation-grade requirements, fake fixture blocking |
| VGGT Public Case Study | evidence-linked outline, missing evidence, no SparseConv3D success claim |
| Public Release RC Gate | full tests, mypy, privacy, quality, demo, docs, no secrets/raw/model files |

## Live / Optional Tests

- Live tests remain skipped by default.
- Export adapters with optional third-party dependencies must have fake/default
  tests first.
- Any live or binary export path must be explicitly opt-in.

## Regression Rules

- Fail if old project naming appears in new v0.7 files.
- Fail if secrets or credential-like values appear.
- Fail if raw data, private paths, or licensed model files are included.
- Fail if fake/demo results are marked observed.
- Fail if unknown plugin code is executed by default.
- Fail if live networking becomes required by default.
