# v0.6.0 Non-goals

Status: scope locked.

Round: 103.

v0.6 remains local-first and review-first. It should not grow into a cloud
product, automatic execution system, or automatic paper author.

## Explicit Non-goals

- No SaaS product.
- No cloud-hosted user system.
- No automatic upload of private research data.
- No automatic final paper conclusion generation.
- No default network access.
- No automatic remote experiment execution.
- No demo result promoted to real result.
- No legacy project naming.
- No broad future sync adapter implementation.
- No unreviewed plugin execution.
- No cross-project evidence transfer without human review.
- No public VGGT case study without real evidence.

## Output Boundaries

- Workspace dashboards are review surfaces.
- Cross-project graphs show reusable patterns, not proof transfer.
- Paper writing scaffolds are outlines and checklists, not final prose.
- Plugin registries are metadata, not automatic trust.
- Benchmark replay is demo/replay material, not a new experiment.

## Safety Boundaries

- Do not package secrets.
- Do not package raw data.
- Do not package private model files.
- Do not include private project links.
- Do not claim SparseConv3D success without evidence.
- Do not hide uncertainty in polished exports.
