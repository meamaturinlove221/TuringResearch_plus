# TuringResearch Plus v0.4.0 Roadmap

Status: post-v0.3 planning.

## Candidate Tracks

1. NAS / SMB Shared Artifact Store.
2. SSH / SFTP Remote Artifact Reader.
3. GitHub Artifact Sync.
4. Modal Run Dashboard.
5. Related Work Positioning refinement.
6. Paper Digest / Three-pass Reading expansion.
7. Advisor Pack PDF / PPTX export.
8. UI or dashboard optional.
9. OCR / MinerU optional track.

## Goals

- Improve controlled cross-machine artifact movement while preserving safety
  and review boundaries.
- Make run ingestion and route status easier to inspect.
- Strengthen paper-reading and related-work workflows without replacing human
  review.
- Add richer advisor-facing exports only after evidence and source hygiene
  gates remain stable.

## Non-Goals

- No default live networking.
- No secret or raw-data sync.
- No automatic final paper writing.
- No automatic VGGT or Modal execution.
- No paywall bypass.
- No OCR heavy pipeline unless explicitly scoped as an optional track.

## Entry Criteria

- v0.3 release-prep tests pass.
- Name integrity and package import tests pass.
- Live adapters remain optional and disabled by default.
- Handoff / pod workflow safety policies remain intact.
- Related-work and ontology outputs remain human-review gated.
