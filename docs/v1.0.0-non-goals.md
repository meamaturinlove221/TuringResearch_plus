# v1.0.0 Non-goals

Status: locked after upstream refresh.

Round: 178 upstream-adjusted scope lock.

This document locks what TuringResearch Plus v1.0 will not attempt to ship.

## Product Non-goals

- No SaaS product.
- No cloud user system.
- No team account system.
- No hosted dashboard.
- No automatic telemetry.

## Research Non-goals

- No automatic real experiment execution.
- No automatic VGGT or Modal run execution.
- No automatic final paper writing.
- No generated final paper conclusions without human review.
- No fake/demo replay treated as observed research evidence.
- No SparseConv3D success claim unless real evidence exists.

## Network And Live Non-goals

- No default networking.
- No live adapter requirement for quickstart.
- No real API key requirement for tests or demos.
- No automatic remote artifact sync.
- No Future Sync Adapter implementation.

## Plugin Non-goals

- No unknown third-party plugin execution.
- No default execute-code plugin permission.
- No secrets access for plugins.
- No claim that policy-level sandboxing is an OS-level sandbox.
- No public plugin marketplace as a stable v1.0 feature.

## Pod / Remote Execution Non-goals

- No remote execution orchestrator.
- No remote command execution.
- No SSH provision.
- No tmux launch.
- No Modal execution.
- No automatic git push.
- No automatic Evidence Ledger write from pod output.
- No bidirectional memory sync.

## Paper Ingestion Non-goals

- No MinerU / heavy PDF fallback in v1.0.
- No heavy PDF ingestion runtime guarantee.
- No automatic extraction that bypasses source hygiene or human review.

## Repository Non-goals

- No automatic creation of split GitHub repositories.
- No automatic external push.
- No physical split of core, paper, artifact, dashboard, or plugins by default.
- No weakening of the flagship main repo.
- No install path change caused by optional split planning.

## Public Positioning Non-goals

- No guarantee of star growth.
- No fake users.
- No fake benchmark claims.
- No fake internship, offer, or adoption association.
- No legal compliance guarantee.
- No old project name reintroduction.

## Stability Non-goals

- No freezing of unstable internals.
- No stable promise for optional PDF/PPTX backend internals.
- No stable promise for future local server dashboard internals.
- No stable promise for future physical split scripts.
