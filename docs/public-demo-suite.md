# Public Demo Suite

Status: v0.5 minimal public-safe demo.

Round 96 adds a fake/demo example suite under `examples/public_demo/`. It is
designed for public walkthroughs without exposing private data or claiming real
research outcomes.

## Included Demo Files

- `README.md`
- `demo_research_intent.md`
- `demo_evidence_ledger.json`
- `demo_artifact_index.md`
- `demo_visual_inventory.md`
- `demo_related_work.md`
- `demo_advisor_pack.md`
- `demo_dashboard.html`

## Demonstrated Capabilities

- Evidence ledger
- Artifact audit
- Visual audit
- Paper method card structure
- Related work positioning
- Route DSL planning boundary
- Failure taxonomy
- Advisor pack
- Static dashboard

## Safety Requirements

- All data must be fake/demo.
- No private project path.
- No raw dataset.
- No private model file.
- No API key or token.
- No planned work written as observed.
- No final paper or experiment claim.

## Usage

Open `examples/public_demo/README.md` or `demo_dashboard.html` locally. No server
or network access is required.
