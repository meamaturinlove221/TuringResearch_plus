# v1.2.0 Non-goals

Status: scope guard.

Round: 230.

v1.2 should make the ecosystem more real without turning TuringResearch Plus
into a cloud product, remote execution system, or automatic research engine.

## Non-goals

- no SaaS product;
- no cloud user system;
- no default networking;
- no automatic experiment execution;
- no automatic final paper writing;
- no automatic child repository creation without human approval;
- no automatic docs deployment without review;
- no remote execution orchestrator;
- no SSH/tmux/Modal execution manager;
- no bidirectional memory sync;
- no default MinerU/heavy PDF fallback runtime;
- no OS-level plugin sandbox guarantee;
- no unknown third-party plugin execution;
- no public plugin marketplace launch by default;
- no private data, raw data, secrets, or restricted model payloads;
- no promise of star growth.

## Explicitly Delayed

- broad physical split of core/paper/artifact/dashboard/plugins;
- remote execution orchestration;
- public plugin marketplace;
- collaboration/cloud account workflows;
- live adapter automation beyond explicit opt-in;
- any feature that needs private VGGT data to demo.

## Allowed Planning And Research

v1.2 may plan or research:

- manual physical split execution for case/examples;
- static docs deployment;
- local dashboard polish;
- Pod Lifecycle Manager local checks;
- Context Return Verifier;
- paper writing beta refinement;
- live web/scholar opt-in polish;
- Apify templates;
- MinerU/heavy PDF fallback research;
- OS-level plugin sandbox research.
