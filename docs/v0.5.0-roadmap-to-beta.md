# v0.5.0 Roadmap to Beta

Status: planning draft.

## Beta Priorities

1. Public Release Hardening
2. Real Advisor PDF / PPTX Optional Adapter
3. Lightweight Dashboard Polish
4. Project Template Packaging
5. Public Demo Expansion
6. VGGT Dogfooding Replay Refresh

## Acceptance Criteria

- Full test suite passes on a clean branch.
- Package metadata and entrypoints are stable.
- Public demo contains no secrets, raw data, private paths, or model files.
- Dashboard is clearly static/review-only.
- Advisor binary export, if added, is opt-in and preserves limitations.
- VGGT replay continues to avoid new-result claims.
- Live adapters remain optional and disabled by default.

## Non-Goals

- No full SaaS UI.
- No user system.
- No cloud deployment.
- No automatic paper writing.
- No automatic VGGT experiment execution.
- No default live remote sync.
