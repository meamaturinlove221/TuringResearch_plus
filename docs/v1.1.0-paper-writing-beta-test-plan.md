## New Tests To Add With Implementation

- beta section assembly preserves section status;
- evidence-linked paragraph candidates require evidence refs;
- unsafe claim guard blocks unsupported claims;
- missing result table guard blocks numeric table generation;
- citation status guard blocks fixture-only citation promotion;
- advisor-to-paper bridge produces actions, not final prose;
- paper outline export includes missing evidence and human review markers.

## Privacy And Safety Tests

Every beta workflow should pass:

- name integrity;
- privacy gate;
- public release hygiene;
- no private path scan;
- no secret scan;
- no raw data scan.

## Acceptance Gate

The beta can move forward only if:

- no final paper is generated;
- no camera-ready text is claimed;
- no fake/demo evidence is marked observed;
- no result value is fabricated;
- no citation is fabricated;
- human review is required.
