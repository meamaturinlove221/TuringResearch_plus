# v1.1.0 Known Limitations

Status: release-candidate limitations.

Round: 229.

## Limitations

- No SaaS.
- No cloud user system.
- Split repos still require manual creation.
- Local server is local-only.
- Paper writing beta requires human review.
- No automatic experiment execution.
- No automatic final paper writing.
- Live tests are skipped by default.
- CI does not publish releases automatically.
- GitHub Actions do not create tags, releases, child repositories, or PyPI
  packages.
- Plugin execution remains restricted.
- Third-party plugins remain disabled unless explicitly reviewed.
- Docs site is local/static; no hosted docs deployment is included.
- Compliance and privacy outputs remain review aids, not legal advice.
- No guarantee of GitHub star growth.

## Claim Boundaries

- Do not claim automatic research completion.
- Do not claim final paper conclusions from beta draft packages.
- Do not claim VGGT or SparseConv3D experiment success without evidence.
- Do not mark planned, fake, or demo output as observed evidence.
