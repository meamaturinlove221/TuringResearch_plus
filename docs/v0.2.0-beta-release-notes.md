# TuringResearch Plus v0.2.0 Beta Release Notes

TuringResearch Plus v0.2.0 beta extends the v0.2 alpha evidence and VGGT
dogfooding foundation with optional live-adapter contracts, citation graph
expansion, collision risk review, run ingestion, and controlled handoff
bundles.

This is still a review-oriented beta. Default workflows run in fake or local
mode and do not require API keys.

## Scope

v0.2.0 beta includes:

- Alpha / Sprint 1 evidence and artifact workflows.
- Sprint 2 route, hard gate, failure, method card, and architecture workflows.
- Beta live-adapter contracts and optional Semantic Scholar live path.
- Citation graph expansion with fake/manual/live-optional modes.
- Paper collision risk detector for conservative positioning.
- Modal / experiment run ingestor for local exported summaries.
- Handoff bundle export/import for controlled cross-machine review packages.

## What Is Not Included

- Default live networking.
- Automatic Modal execution.
- Direct VGGT project modification.
- NAS, SSH, GitHub, or cloud artifact sync.
- OCR heavy pipeline.
- Automatic final paper conclusion generation.
- Claims that SparseConv3D succeeded without evidence-ledger support.

## Release Guidance

Use v0.2.0 beta for local review, dogfooding, and fake-mode integration tests.
Any live adapter result, imported run result, collision risk conclusion, or
handoff bundle should remain human-review gated before promotion.
