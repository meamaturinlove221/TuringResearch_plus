I have been building **TuringResearch Plus**, a local-first Research OS for
evidence-led research workflows.

It connects evidence ledgers, artifact audits, experiment routes, paper review,
advisor packs, dashboards, plugin safety, and privacy gates.

The core principle is simple: planned work is not observed evidence. Fake/demo
workflows run locally by default, live adapters are optional, and human review
is required for research claims.

v1.0 includes a public demo, VGGT public-safe dogfooding case, plugin manifest
safety, MCP config polish, modular monorepo structure, and full regression
gates.

It is not automatic research and it does not write final papers. It is a
reviewable system for keeping research state honest.
