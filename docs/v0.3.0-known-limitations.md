# TuringResearch Plus v0.3.0 Known Limitations

## Live and Web

- live web / Apify disabled by default.
- live tests skipped by default.
- Web content is retrieved, not verified.
- No paywall bypass.
- No restricted or private content fetch.

## Research Claims

- Related work positioning requires human paper review.
- Citation graphs and web fixtures are not complete related work.
- Collision and positioning outputs are review material, not final novelty
  decisions.
- No automatic paper writing.
- No automatic final paper conclusion generation.

## Knowledge Graph

- Ontology graph is not final truth.
- Wikilinks are navigation aids, not evidence.
- Vault graph is lightweight local data, not a graph database or UI.

## Operations

- No NAS / SSH / GitHub Sync yet.
- No full UI.
- No automatic VGGT experiment execution.
- Modal / pod outputs still need explicit artifact export and human review.

## VGGT Boundary

- TuringResearch Plus does not modify VGGT.
- SparseConv3D success is not claimed without evidence-ledger proof.
- The VGGT Research Knowledge Pack is a planning and review artifact, not an
  experiment result.
