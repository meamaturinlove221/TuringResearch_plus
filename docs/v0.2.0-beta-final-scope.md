## Scope Principles

- Default execution remains fake/dry-run and no-network.
- Live adapters are optional and disabled unless explicitly configured.
- Live search results are sources, not human-verified conclusions.
- Modal remains outside direct TuringResearch execution.
- VGGT project files are not modified by TuringResearch.
- Every beta output must serialize to JSON and Markdown.
- Public API changes require contracts, docs, and tests in the same round.

## Alpha Capabilities Beta Must Preserve

- Package imports for `turing_research` and `turing_research_plus`.
- Name integrity tests.
- Sprint 1 and Sprint 2 integration tests.
- Evidence Ledger conservative status semantics.
- Advisor Pack Markdown output.
- Route pack planned / not-executed status until real artifacts exist.
- Paper Method Card human-review boundary.
- Architecture diagram fixture provenance.
- Default no-network test behavior.
