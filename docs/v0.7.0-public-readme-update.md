# v0.7.0 Public README Update

Status: prepared.

The README has been updated for v0.7 public release-candidate review.

## Required Positioning

- TuringResearch Plus is a local-first research OS.
- Default workflows are fake/demo first.
- Live adapters are optional.
- It does not automatically run experiments.
- It does not automatically write final papers.
- Human review is required.
- VGGT is a dogfooding case.
- Sensitive files are excluded.

## Public Safety Language

The README keeps explicit safety language for:

- no default networking;
- no private data upload;
- no unknown plugin execution;
- no fake/demo/planned item promoted to observed;
- no final paper conclusion generation;
- no compliance output treated as legal approval.

## Release Boundary

The README describes v0.7 as public release-candidate review material. It does
not claim publication, PyPI release, or GitHub release.
