## Safety Guarantees

- Fake/demo outputs are not observed evidence.
- Live tests are skipped by default.
- Live SSH/SFTP is opt-in.
- Unknown plugins remain disabled.
- Remote command execution remains rejected by default.
- Experiment execution remains runbook/review-only.
- ARIS features remain deferred.
- Public demo material contains no raw data, private paths, secrets, or
  restricted model payloads.
