# v1.6.0 Non-goals

Status: final non-goals.

Round: 361.

## Non-goals

- no ARIS implementation;
- no default networking;
- no automatic GitHub child repository creation;
- no automatic PyPI publication;
- no automatic tag creation;
- no automatic GitHub release publication;
- no automatic public docs deployment;
- no automatic external push;
- no automatic remote command execution;
- no default SSH/SFTP;
- no unknown plugin execution by default;
- no upload of private data;
- no raw data packaging;
- no restricted model payload packaging;
- no guarantee of public attention, stars, or adoption.

## Claim Boundaries

- Do not claim deployment happened until a human performs and verifies it.
- Do not claim child repositories exist until a human creates and verifies them.
- Do not claim package publication happened until a human publishes and verifies
  it.
- Do not claim live provider success from optional-live smoke planning.
- Do not claim ARIS behavior exists from study docs.
- Do not convert plan, fake, demo, or review-only material into observed
  evidence.
- Do not write screenshot/demo assets as experiment proof.
