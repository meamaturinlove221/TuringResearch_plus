# v0.7.0 Known Limitations

Status: release candidate limitations.

- No SaaS.
- No cloud user system.
- Plugin sandbox is policy-level, not OS sandbox.
- Unknown third-party plugins are disabled.
- PDF/PPTX backend is optional.
- Compliance assistant is not legal advice.
- Paper deep review requires human validation.
- No automatic experiment execution.
- No automatic final paper writing.
- No default live networking.
- VGGT public case study is dogfooding material, not a success report.
- Public demo material is fake/demo only.
- Privacy scans are pattern-based and require human review.
