# v1.1.0 Split README Hardening

Status: completed.

Round: 211.

This report records README hardening for the local split-ready bundles:

- `split_ready/turingresearch-vggt-case`
- `split_ready/turingresearch-examples`

No external repository was created and no external remote was pushed.

## Goal

Make future spoke repository README files useful without letting them compete
with the flagship TuringResearch repository.

## Required README Sections

Both split-ready README files now include:

- repository positioning;
- relationship to the flagship repository;
- flagship link placeholder without a real URL;
- demo-only or case-only boundary;
- no core framework source statement;
- privacy and claim-safety boundary;
- quickstart;
- how to cite or reference the flagship;
- limitations.

## Star Protection

The README files route users back to the main TuringResearch repository for:

- install;
- docs;
- release notes;
- quickstart;
- public API;
- star.

They do not claim to be published repositories and do not include external
GitHub URLs.
