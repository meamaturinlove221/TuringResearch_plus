# TuringResearch Plus v0.2.0 Alpha Known Limitations

The v0.2.0 alpha is intentionally conservative. It is a local planning,
evidence, and fake/dry-run integration milestone.

## Limitations

- TuringResearch Plus does not default to network access.
- Live Semantic Scholar adapter is not complete.
- Real citation graph expansion is not complete.
- OCR heavy pipeline is not complete.
- Modal execution is not performed directly by TuringResearch.
- TuringResearch only generates route packs, controller prompts, and evidence
  management artifacts; it does not replace VGGT experiments.
- Paper Method Card fixtures are not complete paper readings.
- Architecture diagram fixtures require human review.
- Future Sync Adapters remain non-goal.
- Sprint 2 proposed tools remain capsule-local until public MCP contracts are
  explicitly updated.
- SparseConv3D success is not established by the route pack.
- VGGT visual proof remains not-ready unless real board inventory is supplied.

## Human Review Required

Before any public beta or advisor-facing claim, a maintainer must review:

- VGGT-side artifacts.
- Evidence ledger status changes.
- visual board inventory.
- route pack execution results.
- paper method card provenance.
- architecture diagram accuracy.
