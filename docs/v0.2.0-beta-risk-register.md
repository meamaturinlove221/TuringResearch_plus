# TuringResearch Plus v0.2.0 Beta Risk Register

Status: beta scope risk register.

| Risk | Likelihood | Impact | Mitigation |
| --- | --- | --- | --- |
| Live adapter defaults accidentally use network | medium | high | Fake mode default, live marker skip, explicit env opt-in. |
| API keys leak into repo or logs | low | high | `.env.example` only, no-key tests, secret scans before release. |
| Live search results are treated as verified paper understanding | medium | high | Require human-review status and EvidenceRef boundaries. |
| Citation graph expansion exceeds budget or rate limits | medium | medium | BudgetGate, cache, retry, rate-limit policy. |
| Collision detector creates false confidence | medium | medium | Report confidence and unresolved evidence gaps. |
| Modal Run Ingestor is mistaken for Modal executor | medium | high | Ingest local exported packages only; docs repeat no execution. |
| Handoff bundle includes raw data or secrets | medium | high | Manifest denylist and contract tests. |
| SMPL-X body model files accidentally included | low | high | Explicit denylist and bundle tests. |
| Sprint 1/2 regressions | medium | medium | Keep integration tests mandatory. |
| Upstream learning report remains missing | medium | low | Create or explicitly defer before beta implementation starts. |
| Public API drift | medium | medium | Contract-first change rule and namespace integrity tests. |
