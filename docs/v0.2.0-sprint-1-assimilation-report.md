## Reference Comparison Notes

- Upstream Yogsoth repos are skill-first and markdown-first. TuringResearch Plus
  is Python MCP-first, contracts-first, and fake-mode testable.
- Upstream `semantic-scholar-mcp` appears stronger as a live TypeScript MCP
  package. TuringResearch Plus currently has safer fake/default behavior and a
  planned protocol boundary for live adapters.
- Upstream `wiki-vault` has mature package-oriented MCP distribution. TuringResearch
  Plus already implements the local vault primitives and should learn from its
  strict CRUD/query boundary while keeping Python package naming.
- No public upstream scan result changes the Round 36 Sprint 1 Top 5.
