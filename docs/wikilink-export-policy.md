# Wikilink Export Policy

Status: active.

Vault graph Markdown may include inline `[[wikilink]]` labels to make graph
navigation easier. Wikilinks are navigation aids, not evidence.

## Rules

- Wikilinks must preserve the original node label.
- Wikilinks do not imply support or truth.
- Edge type and confidence must remain visible in graph summaries.
- Missing evidence and human-review status must not be hidden.
