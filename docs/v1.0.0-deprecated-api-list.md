# v1.0.0 Deprecated API List

Status: freeze draft.

Round: 176.

No v1.0 API is removed in this round.

## Compatibility Namespace

`turing_research_plus` is not removed. It remains the compatibility namespace
through v1.0.

Future docs may recommend target namespace facades, but old imports must keep
working during v1.0 release hardening.

## Deprecated Later

| API | Status | Replacement | Removal before v1.0 |
| --- | --- | --- | --- |
| direct implementation imports from `turing_research_plus` | deprecated later, not removed | target namespace facades such as `turing_research_core.public_api` | no |

## Deprecation Requirements

Any future deprecation must include:

- replacement path;
- migration note;
- old/new import tests;
- release note entry;
- no default network behavior;
- no plugin execution on import.

## Not Deprecated

The following remain supported in v1.0:

- `import turing_research_plus`;
- new namespace facade imports;
- package metadata imports;
- documented compatibility alias registry.
