# Upstream Academic Output Definition

Round: 392R
Status: gate definition

## Purpose

This document defines what counts as an upstream academic output for future
authorized migration work.

## Accepted Academic Output Evidence

A future academic-output migration must include at least one concrete,
reviewable source item:

- paper PDF;
- arXiv link;
- BibTeX;
- DOI;
- publication page;
- author-provided manuscript.

The migration must also include provenance, authorization status, and
public-sharing boundaries.

## Not Sufficient

The following do not count as academic publication migration by themselves:

- showcase examples;
- summary pages;
- workflow demos;
- migration scaffolds;
- templates;
- rewritten descriptions;
- README positioning;
- public-safe notes without a concrete upstream artifact;
- derived demos without a source academic output.

## Required Metadata

Each accepted academic output should record:

- title;
- author or source owner;
- source type;
- source URL or local source placeholder;
- authorization status;
- public-sharing status;
- license or usage note;
- whether the artifact is copied, linked, summarized, or excluded;
- privacy review status.

## Forbidden Claims

Do not claim that a publication has been migrated unless concrete source
material is present and reviewable.

Do not claim that a showcase, demo, or summary is an academic publication.

Do not promote planned migration work into completed migration status.

## PR #1 Application

PR #1 does not meet this definition. It does not provide concrete publication
materials and is therefore not accepted as academic publication migration.
