# v1.0.0 Demo Expected Output

Status: expected output draft.

Round: 179.

This document lists what a public user should see during the 5-10 minute
quickstart.

## Public Demo Tests

Command:

```powershell
python -m pytest tests/workflow/test_public_demo_suite.py tests/workflow/test_public_demo_expansion.py -q
```

Expected:

- tests pass locally;
- no network access;
- no API key;
- no private project path;
- no live adapter.

## Evidence Ledger

Files:

- `examples/public_demo/demo_evidence_ledger.json`
- `examples/public_demo/projects/*/evidence_ledger.json`

Expected:

- `status` is `demo-only`;
- `requires_human_review` is `true`;
- entries are `planned`, `fake-data`, or `not-enough-evidence`;
- no entry is `observed`.

## Dashboard

Files:

- `examples/public_demo/dashboard/index.html`
- `examples/public_demo/projects/*/dashboard.html`

Expected:

- static HTML;
- demo-only wording;
- no script requirement;
- no remote URL requirement;
- no experiment success claim.

## Advisor Markdown Bundle

Files:

- `examples/public_demo/demo_advisor_pack.md`
- `examples/public_demo/projects/*/advisor_pack.md`

Expected:

- advisor questions or next actions;
- limitations;
- human-review boundary.

## Paper / Related Work Demo

Files:

- `examples/public_demo/demo_related_work.md`

- server name `turingresearch-plus`;
