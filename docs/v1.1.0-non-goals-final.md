# v1.1.0 Non-goals Final

Status: final non-goals.

Round: 205.

v1.1 is not a broad expansion release. It is a post-v1 stabilization and public
surface improvement release.

## Non-goals

- no SaaS;
- no cloud user system;
- no unknown plugin execution;
- no default networking;
- no automatic private data upload;
- no automatic real VGGT experiment execution;
- no automatic Modal experiment execution;
- no OS-level plugin sandbox;
- no large frontend application;
- no automatic final paper generation;
- no guaranteed star growth.

## Also Deferred

- remote execution orchestrator;
- heavy PDF/MinerU fallback as a default path;
- public plugin marketplace;
- broad implementation-module physical split;
- collaboration/cloud account workflows;
- private VGGT data dependent demos;
- automatic GitHub release/tag/publish flow.

## Allowed Only With Explicit Review

- creating `turingresearch-vggt-case`;
- creating `turingresearch-examples`;
- live web adapter improvements;
- CI/CD release automation beyond checks;
- paper writing beta output beyond scaffolds.
