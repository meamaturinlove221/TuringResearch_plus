# v1.3.0 Release Notes

Status: release-candidate notes.

Round: 289.

TuringResearch Plus v1.3.0 is a full original-reference parity
release-candidate line. It turns the v1.2 structural parity work into
fake/default replayable surfaces for Session runtime, Scholar/Web tools,
yogsoth-inspired research catalog workflows, and public demos.

This release does not publish a SaaS product, does not execute experiments
automatically, does not enable live networking by default, does not create child
repositories automatically, and does not adopt ARIS runtime features into v1.3.

- Session runtime parity.
- Context pack runtime.

- Remote return verifier.

## What Is New Since v1.2

- Session parity now has a complete fake pod workflow replay:
  preflight -> context pack -> fake transfer -> fake return -> verifier.
- Scholar/Web parity now exposes complete fake/default tool surfaces and
  walkthroughs.
- MCP parity now maps v1.3 tool surfaces as documentation-contract-only.
- yogsoth parity now includes campaign trace, public dashboards, demos, stress
  library, and convergence decision reports.
- Public demo now has a v1.3 original parity path.
- Security/privacy and full regression now cover v1.3 parity additions.

## What Remains Review-gated

- Release branch push, tag creation, GitHub release, and PyPI publishing.
- Live provider tests and live adapter polish.
- Live SSH/SFTP beyond explicit private opt-in.
- ARIS study and any future cross-model review loops.
- Remote execution orchestration.
- Automatic experiment execution.

## Release Boundary

v1.3.0 remains a local-first Research OS release candidate. Human review is
required before publication, claims, external repository creation, live provider
usage, live SSH/SFTP usage, or public release actions.
