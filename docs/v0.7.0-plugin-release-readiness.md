# v0.7 Plugin Release Readiness

Status: GO WITH REVIEW.

## Ready

- Trusted local plugin loading is metadata-only.
- Plugin sandbox policy blocks forbidden permissions.
- Plugin compatibility harness catches schema, MCP mapping, docs/tests, naming,
  and permission issues.
- MCP distribution docs clarify package name, server name, fake mode, live mode,
  and plugin registry boundaries.
- Safe MCP config example uses blank credential placeholders and plugin-disabled
  defaults.

## Not Ready For Runtime Enablement

- No untrusted plugin execution.
- No dynamic plugin entrypoints.
- No OS sandbox.
- No default live plugin tools.
- No public plugin marketplace publication.

## Required Before Public Runtime Plugins

- Real sandbox design and implementation.
- Plugin source review workflow.
- License and data access review.
- Compatibility matrix for supported plugin versions.
- Maintainer approval for any plugin enablement path.

## Gate Decision

GO WITH REVIEW for metadata-only v0.7 plugin distribution polish.

NO-GO for runtime third-party plugin execution.
