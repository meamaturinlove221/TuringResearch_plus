# TuringResearch Plus Roadmap from v0.2 Beta to v0.3

Round 63 rescopes the v0.3 plan after the latest manual upstream snapshot. The
current detailed plan is `docs/v0.3.0-rescoped-roadmap.md`.

## Rescoped v0.3 Themes

1. Git-based Context Handoff / Pod Workflow
2. Scholar Pipeline Refinement
3. Web Fetch Adapter / Apify Adapter
4. Knowledge Graph / Wiki / Ontology / Skill Routing

## Near-Term Change

Sprint 1 now prioritizes Git-based Context Handoff / Pod Workflow. NAS / SMB /
SSH / GitHub Artifact Sync is deferred as a broad sync track because the
Git-based route is closer to the current dual-machine + VGGT + Modal operating
need and the latest Neocortica-Session signal.

## Still Relevant Later

- Paper Collision Risk refinement
- Related Work Positioning Generator
- Vault Graph Enhancement
- Ontology SOPs
- Modal run dashboard
- richer Advisor Pack export with optional PDF / PPTX

## Non-Goals Until Explicitly Scoped

- Default networking.
- Automatic remote sync.
- Automatic paper conclusions.
- Raw dataset or body model transfer.
- Live execution of VGGT or Modal from TuringResearch Plus.
