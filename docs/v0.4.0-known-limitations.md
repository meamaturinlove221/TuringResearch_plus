# v0.4.0 Known Limitations

Status: integration gate limitations.

## Remote Artifacts

- Live GitHub, SFTP, and cloud paths are optional and disabled by default.
- Remote artifacts are indexed or retrieved, not human verified.
- Proposed imports are not automatically written into Evidence Ledger.
- Large files remain metadata-only unless a future explicit policy changes this.
- Secrets, raw data, and SMPL-X model files remain blocked.

## Dashboard and Run Comparison

- Modal Run Dashboard does not run Modal.
- Dashboard and comparison outputs are not experiment results.
- Run Comparison is metadata/report-level only; it does not inspect pixels or
  images.
- SparseConv3D success still requires real backend evidence.

## Paper Digest and Related Work

- Paper Digest fixtures are not complete paper reviews.
- Related Work Positioning still requires human paper review.
- Collision Risk is conservative and cannot establish definitive no-collision.
- No automatic final paper prose is generated.

## Advisor Export

- Advisor export currently creates Markdown source bundles only.
- PDF, PPTX, DOCX, and HTML exports are future work.
- Figures and tables are listed as source requirements, not generated assets.

## General

- Live remote sync is disabled by default.
- No automatic remote command execution is supported.
- Default tests do not access network.
- Live tests must be explicitly enabled.
- No private VGGT local path is read.
- No NAS/SSH/GitHub/cloud source is treated as verified evidence by default.
- No UI dashboard is included yet.
- No automatic paper conclusions are generated.
- No paywall bypass is supported.
- No raw data / SMPL-X model packaging is supported.
