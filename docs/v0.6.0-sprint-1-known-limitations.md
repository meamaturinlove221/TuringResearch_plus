# v0.6 Sprint 1 Known Limitations

Status: active.

## Workspace

- No SaaS workspace.
- No cloud account system.
- No automatic project discovery outside explicit registry files.
- Workspace registry is not an evidence source.

## Templates

- Generated projects are placeholders.
- Templates do not generate research conclusions.
- Templates do not generate real citations.
- Templates require human review before use.

## Cross-project Graph

- No graph database.
- Pattern extraction is shallow and text based.
- The graph is not final truth.
- Evidence from one project is never applied to another project automatically.
- Shared methods, failures, artifacts, and route patterns are reuse hints only.

## Privacy / Data Policy

- Scanner is pattern based and can have false positives or false negatives.
- No encryption system.
- No permission system.
- No automatic redaction overwrite.
- No automatic deletion.
- Human review remains required.

## Default Runtime

- No network access by default.
- No private VGGT path reads.
- No automatic remote experiment execution.
- No public release yet.
