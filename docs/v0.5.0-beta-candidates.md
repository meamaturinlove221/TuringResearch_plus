## Candidate Notes

- Public release hardening is the strongest beta foundation because alpha now
  has demo, dashboard, template, packaging, and replay surfaces.
- Real PPTX/PDF export is useful, but must remain optional and preserve
  limitations.
- Dashboard refinement should stay lightweight and local-first.
- MCP installation flow should focus on local install and example config, not
  marketplace publishing yet.
- Real VGGT replay with new artifacts should wait until new artifacts are
  actually available.
