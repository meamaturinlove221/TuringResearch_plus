# TulingResearch Plus v0.2.0 Sprint 1 Plan

Round 35 selects Sprint 1 work. This is planning only: no code is implemented,
no network access is used, no real VGGT experiment is run, and no fake result is
produced.

## Sprint Goal

Build the local evidence infrastructure needed for VGGT / SMPL-X human-prior
dogfooding before further experiments or advisor reporting:

- audit local artifacts and bundles
- classify visual evidence levels
- maintain a structured VGGT/SMPL-X evidence ledger
- build advisor-readable packs without overclaiming
- extract paper figures/tables for method understanding

## Selected Items

1. Artifact Auditor
2. Visual Evidence Auditor
3. VGGT/SMPL-X Evidence Ledger
4. Advisor Pack Builder
5. PDF Phase B Figure/Table Extraction

The selected list is detailed in `docs/v0.2.0-sprint-1-selected-items.md`.

## Implementation Order

| Order | Item | Why first/next |
| ---: | --- | --- |
| 1 | VGGT/SMPL-X Evidence Ledger contracts | Defines statuses and prevents overclaiming before other tools write evidence. |
| 2 | Artifact Auditor | Produces trustworthy artifact records for the ledger. |
| 3 | Visual Evidence Auditor | Classifies visual artifacts and advisor-readiness levels. |
| 4 | Advisor Pack Builder | Consumes ledger and visual audit outputs; should not invent evidence. |
| 5 | PDF Phase B Figure/Table Extraction | Builds method-card support for NeuralBody/HumanRAM interpretation. |

This order respects the user-prioritized Top 1-5 while starting with the
contract/status surface that the Top 1-4 items need.
