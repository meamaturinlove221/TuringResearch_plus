# v0.3 Sprint 1 Final Scope

Sprint 1 theme: Git-based Context Handoff / Pod Workflow.

## Goals

- Use Git as the context handoff channel.
- Avoid full session teleport.
- Package durable context as structured files.
- Return pod outputs as auditable artifacts.
- Integrate returned outputs with Run Ingestor, Artifact Auditor, Evidence
  Ledger, Failure Taxonomy, and Advisor Pack.

## In Scope

- `ContextPackage` design.
- `MemoryPolicy` design.
- `PodWorkflowSpec` design.
- `StructuredOutputReturn` design.
- `HandoffSafetyPolicy` design.
- `GitTransportPolicy` design.
- contracts draft for Git context handoff and pod workflow.
- feature capsule skeletons for Sprint 1.

## Out of Scope

- Implementing code.
- Running Git commands.
- Running remote pods.
- Running Modal or RunPod.
- Controlling remote execution.
- Copying upstream scripts.
- Default network behavior.
- Raw data or SMPL-X body model transfer.

## Acceptance Criteria

- Design docs describe package contents and output return package.
- Contracts define stable draft models and policies.
- Safety rules forbid secrets, `.env`, raw data, and body model files.
- Differences from Neocortica-Session are explicit.
- v0.2 Handoff Bundle and Run Ingestor remain the structured base.
