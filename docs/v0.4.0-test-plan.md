# v0.4.0 Test Plan

Status: planning.

## Common Gates

- Default tests do not require network.
- Missing credentials do not fail default tests.
- Live / remote tests are explicitly marked and skipped by default.
- No raw data, secrets, `.env`, private configs, cache folders, or SMPL-X body
  model files are copied.
- Remote artifact retrieval is not marked human verified.
- Planned route outputs are not promoted to observed evidence.

## Feature Test Requirements

### GitHub Artifact Sync

- Fake repository fixture.
- Manifest-only dry-run.
- Secret and raw-data exclusion.
- No push in default tests.
- Proposed evidence updates only.

### SSH / SFTP Remote Artifact Reader

- Fake SFTP listing fixture.
- Credential absence path.
- Read-only path policy.
- No remote command execution.
- Large-file summary-only behavior.

### NAS / SMB Shared Artifact Store

- Local fixture path standing in for shared storage.
- Path traversal rejection.
- Allowlist/denylist behavior.
- Summary-only handling for large files.

### Cloud Object Artifact Index

- Fake bucket listing fixture.
- Index-only output.
- No object download by default.
- Credential absence path.

### Remote Artifact Integration Gate

- Handoff bundle + run ingest + artifact audit + evidence proposed updates.
- No fake result marked observed.
- No SparseConv3D success without evidence.

### Modal Run Dashboard

- Reads `RunIngestReport` fixtures.
- Shows status, missing artifacts, hard gates, and failure categories.
- Does not run Modal.

### Paper Digest / Three-pass Reading

- Manual/cached paper note fixtures.
- Pass 1/2/3 outputs.
- Requires-human-review retained.
- No final paper prose generation.

### Advisor PDF / PPTX Export

- Markdown input fixture.
- Export manifest.
- Limitations and not-ready claims included.
- No invented evidence.
