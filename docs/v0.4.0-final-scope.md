# TuringResearch Plus v0.4.0 Final Scope

Status: scope locked for planning.

Round 79 locks the first v0.4 phase around controlled remote artifact movement,
read-only remote inspection, run visibility, and later advisor/paper outputs.
This is a planning round only; no code is implemented.

## Scope

v0.4 first-stage scope:

1. GitHub Artifact Sync.
2. SSH / SFTP Remote Artifact Reader.
3. NAS / SMB Shared Artifact Store.
4. Cloud Object Artifact Index.
5. Remote Artifact Integration Gate.
6. Modal Run Dashboard.
7. Paper Digest / Three-pass Reading.
8. Advisor PDF / PPTX Export.

## Primary Track

The primary track is Remote Artifact Foundations:

- export/import review bundles without raw data;
- inspect remote artifact indexes without executing remote code;
- preserve sha256, omitted reason, safety warnings, and manual review status;
- feed returned artifacts into run ingest, artifact audit, evidence proposed
  updates, and advisor inputs.

## Secondary Track

The secondary track is Research Communication:

- Modal run dashboard from structured reports;
- paper digest and three-pass reading expansion;
- advisor PDF / PPTX export from existing Markdown, evidence, and limitations.

## Release Boundary

v0.4 must preserve v0.3 evidence discipline. Remote artifacts are retrieved or
indexed, not human verified. Any evidence promotion still requires explicit
ledger review.
