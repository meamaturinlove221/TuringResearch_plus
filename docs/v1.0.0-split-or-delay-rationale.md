# v1.0.0 Split Or Delay Rationale

Status: final rationale.

Round: 188.

## Why Delay Physical Split

Physical split should be delayed for v1.0 because the public launch already has
enough moving parts:

- README finalization;
- quickstart;
- public demo;
- security/privacy audit;
- API stability;
- install sanity;
- release candidate review.

Adding new repositories before launch would multiply README, license,
security, issue-template, release, and maintenance work.

## Why Case / Examples Are Ready-to-create

`turingresearch-vggt-case` and `turingresearch-examples` are the safest future
spokes:

- they are presentation/demo material;
- they do not move core runtime code;
- they can independently show value;
- they can link users back to the flagship;
- dry-run exports already establish a public-safe direction.

They are still not created automatically. They require human approval.

## Why Core / Paper / Artifact / Dashboard / Plugins Stay Put

Core should not be split before the main repo establishes its public story.
Removing core too early would make the flagship feel hollow.

Paper and artifact modules need more API maturity and standalone demo value
before they can carry separate repository responsibility.

Dashboard needs a clearer standalone package and asset story.

Plugins should wait for a real third-party ecosystem and stronger standalone
review flow. The main repo should keep the core plugin framework and safety
policy.

## Star Protection

The v1.0 launch should concentrate attention on the flagship repository. Spoke
repositories can become growth points later, but only if every spoke README
routes stars, issues, install guidance, and product context back to the main
repo.

## Feedback-driven Split

After public launch, split timing should depend on:

- which demos users actually inspect;
- whether examples need independent issue tracking;
- whether the VGGT case study benefits from a standalone narrative;
- whether maintaining docs in one repo becomes noisy;
- whether the flagship README remains clear with or without spokes.
