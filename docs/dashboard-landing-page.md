# Dashboard Landing Page

Status: static/local-first landing page.

Round: 351.

The dashboard landing page turns the v1.5 showcase scope into a small static
HTML page for public demos and interview walkthroughs. It is generated by
`turing_research_plus.ui.showcase_landing` and written to:

- `examples/public_demo/dashboard_showcase/landing.html`

## Content

The landing page includes:

- project pitch;
- quickstart;
- original parity status;
- public demo;
- docs site;
- split repo readiness;
- safety boundary.

## Runtime Boundary

The page is standalone HTML:

- no JavaScript;
- no external assets;
- no analytics;
- no network requirement;
- no live provider call;
- no SSH or SFTP connection;
- no remote command execution;
- no Evidence Ledger mutation.

## Safety Boundary

The landing page is a static product-style view over existing docs and demo
fixtures. It does not claim that live providers ran, split repos exist, public
deployment happened, or planned/demo outputs became observed evidence.

## Source

The renderer exposes:

- `build_showcase_landing_page()`;
- `render_showcase_landing_html()`;
- `write_showcase_landing(path)`.

Use the generated example as review material only.
