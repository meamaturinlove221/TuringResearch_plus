# v1.4.0 Candidates

Status: adjusted candidates.

Round: 291.

The v1.4 candidate set is adjusted from ARIS study prototypes to original repo
production parity. The human scan summary indicates the useful work is still in
the stable Neocortica/yogsoth surfaces: Session runtime usability,
Scholar/Web tool entry points, MCP/SKILL/README clarity, Apify/cache behavior,
and research-engine replay.
