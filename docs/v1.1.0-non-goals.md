# v1.1.0 Non-goals

Status: scope guard.

Round: 199.

v1.1 should be a focused post-launch growth and usability sprint. It should not
reopen the full v1.0 release scope.

## Non-goals

- no SaaS product;
- no cloud user system;
- no default networking;
- no automatic experiment execution;
- no automatic final paper writing;
- no advisor replacement claim;
- no unreviewed third-party plugin execution;
- no public plugin marketplace launch;
- no OS-level sandbox guarantee unless separately researched and proven;
- no remote execution orchestrator;
- no heavy PDF/MinerU fallback as default path;
- no automatic split of core, paper, artifact, dashboard, or plugin framework;
- no child repository URL before the child repository actually exists;
- no private data, raw data, secrets, or restricted model payloads;
- no promise of star growth.

## Explicitly Delayed

- broad physical split of implementation modules;
- remote execution orchestration;
- heavy PDF fallback runtime;
- public plugin marketplace;
- collaboration/cloud account workflows;
- any feature that needs private VGGT data to demo.

## Allowed Planning Work

v1.1 may still plan these areas:

- future split sequencing;
- local server dashboard scope;
- public docs site structure;
- research paper writing beta boundaries;
- case study templates;
- live adapter polish with fake-default tests;
- CI/CD release check improvements.
