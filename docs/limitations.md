# Limitations

Status: public release-candidate limitations.

## Product Limitations

- No SaaS.
- No user account system.
- No hosted cloud dashboard.
- No automatic remote experiment execution.
- No default live networking.
- No automatic final paper writing.

## Research Limitations

- Fake/demo replay is not a benchmark result.
- Public demos are not research evidence.
- VGGT is a dogfooding case, not a success report.
- Paper scaffolds require human review.
- Deep review checklists do not prove complete paper reading.
- SparseConv3D success is not established.

## Plugin Limitations

- Unknown third-party plugin code is not executed.
- Trusted local plugin loading is metadata-first.
- OS-level plugin sandboxing is not implemented.
- Plugin tools are disabled by default unless explicitly reviewed.

## Export Limitations

- PDF/PPTX export backends are optional.
- Missing backends skip with recorded reasons.
- Advisor exports are review reports, not final publication material.
- No charts or figures are fabricated.

## Privacy And Compliance Limitations

- Privacy scanning is pattern-based and can miss issues.
- Compliance reports are checklists and not legal advice.
- License posture requires maintainer review.
- Sensitive files must stay excluded from public examples and release bundles.
