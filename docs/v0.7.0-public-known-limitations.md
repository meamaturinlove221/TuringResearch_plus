# v0.7 Public Known Limitations

Status: public RC limitations.

## Product Scope

- No SaaS.
- No user account system.
- No cloud deployment workflow.
- No automatic remote experiment execution.
- No default live networking.
- No automatic final paper writing.

## Plugin Scope

- Trusted local plugin loading is metadata-first.
- Unknown third-party plugins are not executed.
- OS-level sandboxing is not implemented.
- Plugin capabilities remain disabled by default until explicitly reviewed.

## Export Scope

- PDF/PPTX export backends are optional.
- Missing export backends must skip gracefully.
- Exported advisor material remains review-only.
- No chart, figure, or result value is fabricated.

## Research Scope

- Public demos are fake/demo only.
- VGGT case study is a dogfooding narrative draft, not a success report.
- Paper deep review is a checklist, not completed paper reading.
- Compliance reports are checklists and not legal advice.
- Privacy scanning is pattern-based and not a guarantee.
- SparseConv3D success is not established.

## Release Scope

- Public release requires maintainer approval.
- License posture must be confirmed before publication.
- Dirty worktree state must be resolved before pushing a release branch.
