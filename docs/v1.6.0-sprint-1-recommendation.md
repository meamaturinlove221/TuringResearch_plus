# v1.6.0 Sprint 1 Recommendation

Status: planned recommendation.

Round: 359.

## Recommended Sprint 1

Start v1.6 with **real docs deployment decision support**, not automatic
deployment.

Sprint 1 should produce:

1. GitHub Pages approval checklist;
2. final docs dist scan;
3. URL insertion policy;
4. rollback plan;
5. post-deployment verification plan;
6. dashboard UX polish backlog;
7. split repo final confirmation checklist.

## Why This First

The v1.5 dry-run already produced a deployable static package. A carefully
gated docs deployment is the smallest real public action and gives the project a
clear external proof point without introducing live providers, remote execution,
or ARIS.

## Explicit Deferrals

- physical split repo creation can follow after docs publication decisions;
- optional live E2E can follow after private credential policy review;
- plugin execution remains disabled by default;
- ARIS remains study-only and should not enter Sprint 1.
