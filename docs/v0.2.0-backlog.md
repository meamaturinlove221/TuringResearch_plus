# TulingResearch Plus v0.2.0 Backlog

Round 31 breaks the v0.2.0 roadmap into executable backlog items. This document is planning only and does not authorize implementation in this round.

## Backlog Rules

- Do not make live network access the default path.
- All live adapters must be optional and adapterized.
- Live tests must be marked `live` or `manual` and skipped by default.
- All public tool changes start in contracts.
- All artifacts must preserve EvidenceRef where applicable.
- Source Hygiene remains required for public or upstream-derived material.
