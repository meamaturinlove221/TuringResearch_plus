   - "This is TuringResearch Plus, a local-first Research OS."
2. Show public demo.
   - "Everything here is fake/demo first and privacy-safe."
3. Show evidence/artifact/route.
   - "The system separates planned work from observed evidence."
4. Show paper/dashboard/advisor.

   - "Plugins are manifest-first and unknown code is not executed."

1. Problem framing: scattered research state.
2. Architecture overview: workspace, evidence, artifact, route, paper,

3. Demo path: public demo, dashboard, advisor pack.
4. Dogfooding: VGGT case as workflow example, not success proof.
