# v0.7 Plugin Known Limitations

Status: integration-gate limitations.

## Current Limits

- Plugins are still metadata-first.
- Unknown third-party plugin code is not executed.
- Dynamic Python entrypoints are not loaded.
- The sandbox policy is not an OS-level sandbox.
- MCP plugin registry entries are not enabled public MCP tools.
- Compatibility reports do not approve runtime execution.
- Live plugin behavior remains opt-in and future-scoped.
- Plugin registry promotion requires human review.

## Distribution Limits

- No PyPI release is performed by this gate.
- No GitHub release is created by this gate.
- `.mcp.example.json` is a safe template, not a private live config.
- Real credentials must stay outside the repo.

## Safety Limits

- Safety checks are static policy checks, not legal or security guarantees.
- Reviewers must still inspect plugin source, license, data access, and runtime
  behavior before any future enablement.
- `execute_code`, `shell_access`, `secrets_access`, and `remote_write` remain
  blocked by default.
