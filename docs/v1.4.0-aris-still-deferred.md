# v1.4.0 ARIS Still Deferred

Status: deferred.

Round: 291.

ARIS remains valuable as a future reference, but it is still not part of the
v1.4 implementation line. Round 291 redirects v1.4 toward original repo
production parity.

## Not Implemented in v1.4

- ARIS runtime;
- cross-model review;
- proof-checker;
- meta-optimize;
- paper-claim-audit;
- automated sleep research loop;
- session stop hook;
- autonomous paper-writing or resubmission machinery.

## Why Deferred

- v1.3 created broad fake/default parity across the original references.
- v1.4 should make those surfaces production-like before adding a new research
  loop family.
- ARIS-style review risks false authority if it arrives before production
  evidence and regression boundaries are tighter.
- Production parity work is more directly aligned with the latest human scan.

## Re-entry Condition

ARIS can be reconsidered only after v1.4 production parity gates are green and a
new scope lock explicitly approves an ARIS study or prototype lane.
