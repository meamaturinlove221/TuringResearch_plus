# v1.3.0 Upgrade Guide

Status: release-candidate upgrade guide.

Round: 289.

## Version

The v1.3 release-candidate version is `1.3.0rc0`.

## Install

Use the main repository as the install target:

```bash
python -m pip install -e .[dev]
```

For MCP-oriented local checks:

```bash
python -m pip install -e .[dev,mcp]
```

## Import Paths

The compatibility namespace remains available:

```python
import turing_research_plus
```

Facade namespaces remain importable:

```python
import turing_research_core
import turing_research_paper
import turing_research_artifact
import turing_research_experiment
import turing_research_dashboard
```
