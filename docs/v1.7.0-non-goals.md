# v1.7.0 Non-goals

Status: planned non-goals.

Round: 389.

v1.7 does not default to broad new feature implementation. It focuses on
post-v1.6 public release follow-through and review.

## Non-goals

- No automatic public deployment.
- No fake public URL.
- No automatic Git tag creation.
- No automatic GitHub release publication.
- No automatic PyPI publication.
- No automatic split repository creation.
- No automatic external push.
- No default live networking.
- No default SSH/SFTP.
- No remote command execution by default.
- No private data upload.
- No unknown plugin execution by default.
- No analytics by default.
- No automatic evidence-ledger mutation.
- No automatic experiment execution.
- No automatic final paper generation.
- No ARIS implementation by default.
- No cross-model review runtime.
- No proof-checker runtime.
- No meta-optimize runtime.
- No paper-claim-audit runtime.

## Naming Boundary

Public docs should continue to use TuringResearch. Compatibility package and
import names should remain stable until a package migration decision is made.
