# v0.7.0 Implementation Order

Status: scope locked.

## Default Order

1. Trusted Local Plugin Loading.
2. Plugin Sandbox Policy.
3. Plugin Compatibility Test Harness.
4. MCP Distribution Polish.
5. Dashboard Refinement.
6. Real Advisor PDF Export.
7. Real Advisor PPTX Export.
8. Dataset / License Compliance Assistant.
9. Local-first Research Vault UI.
10. Paper Deep Review Mode.
11. VGGT Public Case Study Builder.
12. Public Release RC Gate.

## Dependency Logic

- Trusted plugin loading must stay local and disabled until sandbox policy is
  explicit.
- Sandbox policy should land before compatibility harness promotes any runtime
  behavior.
- MCP distribution polish can proceed once plugin boundaries are stable.
- Dashboard refinement can consume existing v0.6 reports without needing a
  service.
- PDF/PPTX export adapters should build on advisor export manifests and remain
  optional.
- Dataset/license compliance should land before broader public demo expansion.
- Vault UI and paper deep review should stay local-first and review-gated.
- VGGT public case study should wait for case-study safety and evidence review.
- Public Release RC Gate must run after all selected v0.7 surfaces are frozen.

## Checkpoint Strategy

- Push after each long implementation round only from a clean, correct branch.
- Use checkpoint commits if a long round crosses 50 percent completion and has
  many coherent changes.
- Never push from a dirty mixed-history branch.
