# v1.0.0 Public Demo Walkthrough

Status: public demo walkthrough draft.

Round: 184.

This walkthrough explains how to inspect the public demo without private data,
network access, API keys, or real VGGT material. It does not add functionality
or generate research conclusions.

## Boundary

Everything in this walkthrough is demo only.

- No real experiment is executed.
- No live adapter is enabled.
- No final paper conclusion is generated.
- No unknown plugin is executed.
- Demo evidence is not observed research evidence.
- Human review remains required.

## 1. Workspace

Start with:

- `examples/public_demo/README.md`
- `examples/public_demo/workspace_demo/workspace_overview.md`
- `examples/public_demo/workspace_demo/workspace.yaml`

The workspace shows three fake projects:

- VGGT-like demo;
- paper survey demo;
- software tooling demo.

Expected takeaway: workspace views organize projects, but they are navigation
aids, not evidence sources.
