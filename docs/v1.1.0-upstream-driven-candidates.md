## Entry Criteria

- contract draft;
- fake/default workflow test;
- privacy and security review;
- no default networking;
- no copied upstream code;
- human review boundary in docs.

## Non-goals For v1.1 Candidates

- no OS-level sandbox guarantee;
- no remote execution orchestration by default;
- no heavy PDF fallback as default path;
- no final research conclusion generation.
