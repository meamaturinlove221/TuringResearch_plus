# v1.1.0 More Public Demo Cases

Status: implemented fake/demo cases.

Round: 223.

The v1.1 public demo suite now includes four additional public-safe demo cases
so TuringResearch does not look like a VGGT-only tool. All cases use fake/demo
data and require human review.

## Added Cases

| Case | Purpose | Boundary |
| --- | --- | --- |
| `robotics_paper_survey_demo` | Show paper survey review for robotics manipulation | Demo-only paper notes |
| `medical_imaging_experiment_demo` | Show experiment planning without patient data | No real patient data or medical advice |
| `software_tooling_research_demo` | Show tooling research without private logs | No telemetry or private logs |
| `multimodal_model_eval_demo` | Show evaluation planning without benchmark claims | No real benchmark results |

## Required Files Per Case

- `README.md`
- `north_star.md`
- `evidence_ledger.json`
- `artifact_index.md`
- `related_work.md`
- `route_plan.md`
- `advisor_pack.md`
- `dashboard_data.json`
- `privacy_note.md`

## Safety Boundary

- Fake/demo only.
- No secrets.
- No raw data.
- No real patient data.
- No private dataset.
- No real private logs.
- No unsupported claims.
- No observed result status.
- Human review required.
