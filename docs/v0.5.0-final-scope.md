# v0.5.0 Final Scope

Status: scope locked.

Round: 91.

## Goal

v0.5 focuses on productization, local usability, advisor export, project
templating, packaging polish, public demos, and careful VGGT dogfooding replay.

## Selected Scope

1. Lightweight Web UI / Dashboard
2. Real Advisor PDF / PPTX Export
3. Project Template Generator
4. MCP / CLI Packaging Polish
5. Public Demo Suite
6. Real VGGT Dogfooding Replay

## Product Direction

The release should make v0.4's review-first artifacts easier to inspect and
share without changing evidence discipline. UI and export work must preserve
limitations, source refs, and human-review status.

## Non-Goals

- No complex SaaS.
- No user system.
- No cloud deployment.
- No automatic paper writing.
- No unauthorized data upload.
- No legacy project naming.
- No default live remote sync.
- No automatic VGGT experiment execution.
