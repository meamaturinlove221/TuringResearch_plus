# TuringResearch Plus Upstream Watch

TuringResearch Plus tracks selected public upstream repositories as a planning
input for Race Mode. The watcher records public metadata baselines and later
diffs them to detect new repositories, changed Markdown, changed packages,
skills, MCP tool hints, source modules, tests, and docs.

## Safety Boundary

- Do not modify upstream repositories.
- Do not copy incompatible upstream code.
- Do not implement live auto-sync.
- Do not implement Handoff, NAS, SSH, GitHub Artifact Sync, or cloud artifact
  sync in this round.
- Do not treat first-baseline entries as new features.
