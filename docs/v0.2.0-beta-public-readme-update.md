# Public README Update Notes for v0.2.0 Beta

The README should describe v0.2.0 beta as a local-first, evidence-disciplined
research workflow engine rather than a fully automated research system.

## Suggested Additions

- Mention optional live Semantic Scholar support with fake mode as the default.
- Add Citation Graph Expansion under paper intelligence.
- Add Paper Collision Risk Detector as conservative positioning support.
- Add Modal / Experiment Run Ingestor as a local summary importer, not a Modal
  executor.
- Add Handoff Bundle Export / Import as controlled review-package transfer, not
  remote sync.
- Clarify that v0.2 beta does not bundle raw data, secrets, or SMPL-X body model
  files.

## Wording Boundaries

- Do not claim automatic related-work completion.
- Do not claim SparseConv3D success.
- Do not describe handoff as NAS, SSH, GitHub, or cloud sync.
- Do not imply live results are human-verified.
