# v0.7.0 Upgrade Guide

Status: release candidate upgrade guide.

## From v0.6 RC

Update version metadata to `0.7.0rc0` and reinstall locally:

```powershell
python -m pip install -e .[dev]
python -m pytest -q
```

## What Changed

- Plugin review moved beyond manifest-only planning into trusted local metadata
  loading, sandbox policy, and compatibility reports.
- Dashboard and advisor export surfaces gained refined static views, optional
  PDF/PPTX export paths, and export quality gates.
- Compliance, vault UI, paper deep review, public case study, and public demo
  workflows are now part of the public release-candidate surface.

## What Did Not Change

- Live adapters remain optional.
- Unknown third-party plugins remain disabled.
- PDF/PPTX backends remain optional.
- Compliance reports are not legal advice.
- Paper review still requires human validation.
- Experiments are not executed automatically.

## Recommended Verification

```powershell
python -m pytest -q
python -m mypy src
python -m pytest tests/workflow/test_v0_7_full_fake_replay.py -q
python -m pytest tests/contract/test_v0_7_security_privacy_gate.py -q
```
