## Current Positioning

TuringResearch Plus is a local-first research OS. It is fake/demo first by
default, with optional live adapters disabled unless explicitly enabled. It
does not automatically run experiments, execute unknown plugins, upload private
data, or generate final paper conclusions.

Current public-entry status:

- v1.0 release-candidate docs are archived for human review;
- v1.1 scope is locked around post-v1 stabilization and public-surface polish;
- `split_ready/` contains local export bundles only, not published child
  repositories;
- the main repository remains the only install and quickstart entry.

## Safety Defaults

- Demo examples are not research results.
- VGGT is a dogfooding case, not a success report.
- Live features are optional.
- Sensitive files are excluded.
- Planned work must not be described as observed evidence.
- Human review is required.
