# Compliance Disclaimer

Status: v0.7 minimal policy.

The dataset / license compliance assistant is a research checklist tool. It is
not legal advice, does not determine legal rights, and does not replace review
by project owners, institutional counsel, dataset providers, model licensors, or
publication venues.

## Boundaries

- No license text is downloaded automatically.
- No private or restricted data is packaged.
- No legal conclusion is generated.
- Unknown or restricted licenses require human review.
- Public release decisions remain maintainer decisions.

## Use

Use the compliance report to record known license metadata, missing license
information, use restrictions, redistribution risk, publication risk, and
review blockers before sharing a project, case study, advisor pack, paper draft,
or demo package.
