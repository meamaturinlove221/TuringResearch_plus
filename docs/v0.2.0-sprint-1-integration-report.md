Round 42 checks the Sprint 1 data flow across the local VGGT / SMPL-X evidence
stack, advisor pack outputs, and PDF Phase B paper asset path.

This is an integration gate. It does not add live APIs, does not read `D:/vggt`,
does not run VGGT experiments, does not generate PPTX/PDF, and does not
implement Future Sync Adapters.
