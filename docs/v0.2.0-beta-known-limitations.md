# TuringResearch Plus v0.2.0 Beta Known Limitations

## Live and Network Limits

- Live API access is disabled by default.
- Live tests are skipped by default.
- Default tests and workflows do not use network access.
- Live Semantic Scholar is optional and requires explicit opt-in.
- Live results are retrieved metadata, not human-verified evidence.
- Citation graph output is not human-verified related work.
- Real citation graph expansion is not a substitute for human related-work review.

## VGGT and Modal Limits

- TuringResearch Plus does not run VGGT experiments.
- TuringResearch Plus does not run Modal jobs.
- Run ingest reads exported summaries or fixtures only.
- SparseConv3D success is not claimed unless an evidence ledger has real
  supporting evidence.

## Artifact and Handoff Limits

- Handoff bundles are local review packages, not sync infrastructure.
- Handoff bundles exclude raw data and SMPL-X model files.
- NAS, SMB, SSH, SFTP, GitHub artifact sync, and cloud storage are not implemented yet.
- Raw data, secrets, cache folders, and SMPL-X body model files are not bundled.
- Import produces proposed updates only and does not overwrite local ledgers.

## Paper Intelligence Limits

- Paper method cards built from fixtures are not complete paper readings.
- Collision risk reports are conservative and require manual review.
- Safe claims are limited to current evidence and should not be treated as final
  novelty claims.
- No automatic paper conclusion generation is included.

## Product Limits

- No publishing or release tagging is performed by beta integration.
- No final paper conclusions are generated.
- No large UI is included.
- OCR heavy pipeline is not included.
