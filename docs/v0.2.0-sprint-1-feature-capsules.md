## Future Sync Adapter Boundary

Sprint 1 does not implement:

- Handoff Bundle Export / Import
- NAS / SMB Shared Artifact Store
- SSH / SFTP Remote Artifact Reader
- GitHub Artifact Sync
- cloud object storage adapters

Capsules may keep future compatibility notes, but they must not implement
cross-machine synchronization.
