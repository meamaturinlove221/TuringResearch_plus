# v0.6.0 Public README Update Notes

Status: release prep notes.

The top-level README now describes TuringResearch Plus as a local,
MCP-first, review-first research workflow engine. It avoids release marketing
claims and keeps the boundaries clear for public readers.

## README Positioning

- The package is local and review-first.
- Default tests use fake services, local fixtures, and dry-run workflows.
- Live adapters are optional and disabled by default.
- TuringResearch Plus does not run experiments or write final paper
  conclusions by itself.
- VGGT is documented as a research dogfooding case, not proof of new
  experiment results.

## Safety Text Kept Explicit

- No default network access.
- Source hygiene blocks unsafe or unauthorized source material.
- No real API keys or live network access are required for default tests.
- No raw data or SMPL-X body model packaging.
- No automatic remote command execution.
- No fake/demo/planned item promoted to observed.
- No SparseConv3D success claim without real evidence.

Round 125 updates prepared version metadata to `0.6.0rc0`. This marks release
candidate preparation, not a published release.
