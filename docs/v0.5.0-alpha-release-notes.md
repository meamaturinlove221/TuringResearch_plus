# TuringResearch Plus v0.5.0 Alpha Release Notes

Status: release preparation complete.

Release date: 2026-05-21.

v0.5.0 alpha focuses on productization and public-readiness scaffolding. It
adds a static dashboard, advisor PDF/PPTX export planning, project templates,
packaging polish, a public demo suite, and a review-only VGGT dogfooding replay.

## Added

- Lightweight Dashboard UI for static HTML / Markdown review dashboards.
- Advisor PDF / PPTX Export Plan with templates and manifest, without binary
  export by default.
- Project Template Generator for standard research project skeletons.
- MCP / CLI Packaging Polish with `.mcp.example.json`, `.env.example`, CLI docs,
  MCP docs, and packaging contract tests.
- Public Demo Suite with fake/demo evidence, artifact, visual, related-work,
  advisor, and dashboard examples.
- Real VGGT Dogfooding Replay report that stitches existing local review
  artifacts into an end-to-end replay chain.
- v0.5 Alpha Integration Gate for fake/default workflows.

## Changed

- Package version advanced to `0.5.0a0`.
- v0.5 docs now separate static UI, plan-only export, project templates,
  packaging readiness, public demo material, and replay boundaries.

## Fixed

- Added v0.5 alpha fake end-to-end tests.
- Added v0.5 alpha contract checks.
- Added packaging tests for CLI/MCP entrypoints and no-secret examples.
- Added public demo tests that reject private paths, model files, and token-like
  values.

## Known Limitations

- Dashboard is static / lightweight.
- No full SaaS UI is included.
- No real PPTX/PDF generation by default.
- No automatic experiment execution.
- No automatic paper conclusion generation.
- Live adapters remain optional.
- Remote sync remains optional and safety-gated.

## Release Boundary

v0.5.0 alpha is suitable for local fake/default review workflows and maintainer
review. It is not a PyPI release, not a GitHub release, not a real VGGT
experiment result, and not a binary advisor export implementation.
