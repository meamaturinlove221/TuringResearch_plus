# TuringResearch Plus v0.2.0 Sprint 1 Non-Goals

Sprint 1 is evidence infrastructure for local VGGT dogfooding. It is not a live
research-result production sprint.

## Explicit Non-Goals

| Non-goal | Status | Reason |
| --- | --- | --- |
| Run real VGGT experiments | requires-real-experiment | Sprint 1 audits and packages evidence; it does not launch training or inference. |
| Modify VGGT files | missing | No write path into VGGT is selected. |
| Claim V120/V121 success | requires-human-review | Local scan did not confirm V120/V121 or Modal artifacts. |
| Make promotion decisions | requires-human-review | Review-ready proxy is not promotion. |
| Generate fake paper results | fake-data | Fake data is allowed only in tests and must be labeled as fake. |
| Live Semantic Scholar adapter | planned | Useful later, but Sprint 1 stays local/manual to avoid live-network dependency. |
| Citation graph real expansion | planned | Requires live adapter readiness and budget controls; defer to later sprint. |
| Vault graph enhancement | planned | Useful, but not the immediate VGGT bottleneck. |
| Upstream watch real snapshot diff | planned | Defer until local evidence surfaces are stable. |
| Handoff / NAS / SSH / GitHub sync | planned | Future sync adapters belong to v0.3+ and should not be implemented in Sprint 1. |

## Rejected Or Deferred Candidates

| Candidate | Decision | Reason |
| --- | --- | --- |
| Modal Experiment Run Ingestor | folded into Artifact Auditor | Promote only if a later local scan confirms V120/V121 or Modal artifacts. |
| Live Semantic Scholar adapter | rejected for Sprint 1 | No live networking in this sprint; use manual/schema fallback. |
| Citation graph real expansion | rejected for Sprint 1 | Depends on live adapter and budgeted expansion. |
| Vault graph enhancement | rejected for Sprint 1 | Valuable after evidence ledger and artifact auditor exist. |
| Upstream watch real snapshot diff | rejected for Sprint 1 | Not urgent for local VGGT dogfooding. |
| Handoff Bundle Export / Import | future | v0.3 candidate after local evidence bundle schema stabilizes. |
| NAS / SMB Shared Artifact Store | future | v0.4 candidate after local artifact schema stabilizes. |
| SSH / SFTP Remote Artifact Reader | future | v0.5 candidate; requires remote threat model. |
| GitHub Artifact Sync | future | v0.6 candidate; requires artifact privacy policy. |
| Cloud object storage adapter | future | Requires explicit future adapter lane and user approval. |
