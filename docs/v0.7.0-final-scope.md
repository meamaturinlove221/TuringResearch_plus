# v0.7.0 Final Scope

Status: scope locked.

Round: 127.

v0.7 focuses on controlled public readiness after the v0.6 release candidate:
trusted local plugin loading, explicit sandbox policy, plugin compatibility
tests, MCP distribution polish, dashboard refinement, optional real advisor
exports, dataset/license review assistance, local-first vault UI planning,
paper deep review, VGGT public case study preparation, and public release RC
gating.

This is a scope lock, not an implementation round.

## Release Posture

v0.7 should remain local-first and review-first. The release may add richer
optional surfaces, but defaults must remain fake/demo, no-network, no private
data upload, and no unknown plugin execution.

## Acceptance Gates

- All v0.7 feature capsules exist with FEATURE, contract, SKILL, SOP, and test
  plan files.
- Default tests do not require network access.
- Plugin loading remains trusted-local only and never unknown third-party by
  default.
- Export adapters remain optional and source-bundle compatible.
- Dataset/license assistant outputs review reports, not legal conclusions.
- VGGT case study cannot claim experiment success without evidence ledger
  support.
- Public release gate blocks secrets, raw data, private paths, model payloads,
  fake observed results, and old project naming.

## Human Review Required

- Plugin permission approval.
- Sandbox policy approval.
- Export dependency and binary output review.
- Dataset/license interpretation.
- Citation-grade related-work review.
- VGGT case study claims.
- Public release approval.
