## Highest-risk Decisions

- Public plugin ecosystem timing.
- Any cross-project evidence comparison.
- Any writing assistant output that resembles final paper prose.
- Any public VGGT case study wording.

## Required Review Gates

- Privacy review before workspace exports.
- License review before public demo or plugin examples.
- Evidence review before any cross-project summary.
- Maintainer review before MCP registry or marketplace packaging.
