# v1.5.0 ARIS Still Deferred

Status: scope locked deferred.

Round: 331.

ARIS remains a future reference. v1.5 does not implement ARIS by default. The
earliest recommended ARIS return path is a separately scoped v1.6 study-only
track, not a full merge.

Round 331 reconfirms this boundary as part of the v1.5 Public Externalization
scope lock.

## Not Implemented in v1.5

- ARIS runtime.
- Cross-model review.
- Proof-checker.
- Meta-optimize.
- Paper-claim-audit.
- Automated sleep research loop.
- Session stop hook.
- Autonomous paper-writing or resubmission machinery.

## Why Deferred

- v1.4 completed original repo production parity in the fake/default lane.
- v1.5 should make that work easier to publish, review, split, and package.
- ARIS-style review loops risk false authority if introduced during release
  infrastructure work.
- Any ARIS study needs its own scope lock, safety model, and regression gate.

## Re-entry Condition

ARIS can return only after v1.5 public release and split/deployment decisions
are stable, and only if a later round explicitly scopes a study or fake
prototype. No ARIS behavior should be enabled by default.

## v1.5 Blocked ARIS Work

- no implementation by default;
- no cross-model review;
- no proof-checker;
- no meta-optimize;
- no paper-claim-audit;
- no automated research loop;
- no default evidence promotion from ARIS-like reports.
