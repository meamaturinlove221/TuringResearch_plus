# TuringResearch Plus v0.2.0 Beta Non-Goals

Beta keeps the alpha safety boundary. The following are explicitly out of
scope unless a later planning round reopens them.

- No default network access.
- No live tests running by default.
- No large UI.
- No OCR heavy pipeline.
- No cloud object storage adapter.
- No automatic Modal execution.
- No direct VGGT project modification.
- No live search result written as human-verified evidence.
- No replacement for human paper reading.
- No automatic final paper conclusions.
- No NAS / SSH / GitHub sync beyond reviewed Handoff Bundle Export / Import.
- No raw VGGT data, secrets, or SMPL-X body model files in handoff bundles.
- No public API promotion without contracts, docs, and tests.

## Boundary Reminder

TuringResearch Plus may prepare routes, ingest exported artifacts, and manage
evidence. It must not become an unreviewed remote executor for VGGT.
