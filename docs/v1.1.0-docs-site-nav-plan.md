## Link Policy

- Link only to local repository files.
- Do not write deployment URLs before deployment exists.
- Do not link to nonexistent split repositories.
- Keep source docs as the canonical reference.
- Prefer relative paths so local browsing works.

## Safety Notes

Docs-site pages should preserve:

- fake/demo boundary;
- live adapters disabled by default;
- privacy-first public material;
- no automatic research claim;
- no automatic final paper claim;
- no VGGT or SparseConv3D success claim without evidence.
