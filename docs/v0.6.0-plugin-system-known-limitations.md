# v0.6 Plugin System Known Limitations

Status: review limitations.

## Limitations

- Plugin architecture is manifest-only.
- MCP Plugin Registry does not start a real MCP server.
- Third-party plugins are disabled by default.
- Extension Safety Gate validates declarations only.
- Capability Manifest is a static catalog, not live discovery.
- Skill Marketplace is local documentation only.
- No plugin marketplace publishing.
- No remote install.
- No dynamic Python imports.
- No sandbox runtime.
- Human review is required before enabling any extension.

## Non-claims

- A valid manifest does not mean a plugin is trusted.
- A catalog entry does not mean a tool is public MCP API.
- A safety report does not grant runtime permissions.
- Demo plugin fixtures are not live integrations.
