## Rules

- Capsules do not implement complex business logic.
- Capsule-local proposed tools are not public API until root contracts and
  `docs/mcp-tools.md` are updated.
- No capsule reads missing VGGT local files by default.
- No capsule runs VGGT or fabricates VGGT experiment results.
- No capsule implements Future Sync Adapters in Sprint 1.

## Files

Each capsule contains:

- `FEATURE.md`
- `contract.yaml`
- `SKILL.md`
- `sop.mmd`
- `test_plan.md`
