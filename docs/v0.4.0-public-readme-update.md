# v0.4.0 Public README Update Notes

Status: draft for maintainer review.

Suggested README additions:

- TuringResearch Plus v0.4.0 adds remote-artifact-aware review workflows.
- Remote Artifact Integration normalizes GitHub, SSH/SFTP, NAS/SMB mounted
  paths, and cloud-object-style indexes into review-first artifact references.
- Modal Run Dashboard and Run Comparison present ingested run metadata without
  executing Modal or VGGT.
- Paper Digest expands the three-pass reading scaffold without claiming complete
  paper review.
- Advisor Markdown Export Bundle prepares source material for future PDF/PPTX
  export without generating binary files.

Suggested README limitations:

- Live remote sync is disabled by default.
- Live tests are skipped by default.
- Remote artifacts are not verified evidence.
- No remote command execution is performed.
- No Modal execution is performed.
- No PPTX/PDF export is generated in v0.4.
- No raw data or SMPL-X model files are packaged.
