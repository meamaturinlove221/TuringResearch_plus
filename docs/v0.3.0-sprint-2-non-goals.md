# v0.3.0 Sprint 2 Non-Goals

Sprint 2 explicitly does not include:

- default networking;
- login or paywall bypass;
- restricted content fetching;
- heavy OCR;
- NAS / SMB sync;
- SSH / SFTP remote artifact reading;
- GitHub Artifact Sync;
- large UI;
- Modal Run Dashboard;
- PPTX/PDF Advisor Pack export;
- automatic final paper conclusions;
- automatic human verification of web content;
- old project naming.

Live web / Apify behavior, if implemented later in the sprint, must remain
explicitly opt-in with live tests skipped by default.
