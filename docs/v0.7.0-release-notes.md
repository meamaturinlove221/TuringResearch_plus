# TuringResearch Plus v0.7.0 Release Notes

Status: release candidate prepared.

v0.7 expands TuringResearch Plus from v0.6 review infrastructure into a more
public-release-ready, local-first research OS surface. It improves plugin
review, dashboard/export review, compliance/vault/case-study workflows, and
public demo coverage.

This release candidate is still fake/demo first and review-first. It does not
publish, run experiments, enable unknown plugins, or generate final paper
conclusions.

## Highlights

- Trusted Local Plugin Loading.
- Plugin Sandbox Policy.
- Plugin Compatibility Test Harness.
- MCP Distribution Polish.
- Dashboard Refinement.
- Real Advisor PDF Export with optional backend skip.
- Real Advisor PPTX Export with optional backend skip.
- Export Quality Gate.
- Dataset / License Compliance Assistant.
- Local-first Research Vault UI.
- Paper Deep Review Mode.
- VGGT Public Case Study Builder.
- Public Demo Expansion.

## Release Candidate Decision

The v0.7 public RC gate is `GO WITH REVIEW`. Maintainer review is still required
for branch cleanliness, license posture, package metadata, release notes, and
public example posture.

## Safety Defaults

- Local-first by default.
- Fake/demo first.
- Live adapters optional.
- Unknown third-party plugins disabled.
- Human review required.
- Sensitive files excluded.
