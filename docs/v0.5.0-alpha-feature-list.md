# v0.5.0 Alpha Feature List

Status: release prep.

## Features

1. Lightweight Dashboard UI
2. Advisor PDF/PPTX Export Plan
3. Project Template Generator
4. MCP / CLI Packaging Polish
5. Public Demo Suite
6. Real VGGT Dogfooding Replay

## Lightweight Dashboard UI

Generates local static HTML / Markdown dashboards from existing review
artifacts. It does not run a server, require login, deploy cloud services, or
claim experiment execution.

## Advisor PDF/PPTX Export Plan

Generates plan-only PDF/PPTX export files:

- `advisor_pdf_export_plan.md`
- `advisor_pptx_outline.md`
- `export_manifest.yaml`
- `slide_section_mapping.md`

No real binary export is generated by default.

## Project Template Generator

Generates a standard research skeleton with README, docs, lanes, examples,
contracts, and feature capsule folders.

## MCP / CLI Packaging Polish

Documents and tests package metadata, CLI entrypoints, MCP entrypoint, example
environment configuration, and fake-default behavior.

## Public Demo Suite

Provides fake/demo examples for evidence ledger, artifact audit, visual audit,
paper method card structure, related-work positioning, route DSL, failure
taxonomy, advisor pack, and dashboard.

## Real VGGT Dogfooding Replay

Replays the existing VGGT review chain from research intent to next action
without running VGGT, Modal, or private local paths.
