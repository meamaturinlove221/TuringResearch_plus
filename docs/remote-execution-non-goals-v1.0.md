# Remote Execution Non-goals For v1.0

Status: active v1.0 boundary.

TuringResearch Plus v1.0 does not become a remote execution orchestrator.

## Explicit Non-goals

- No remote command execution.
- No tmux launch.
- No SSH provision.
- No Modal execution.
- No automatic git push.
- No automatic Evidence Ledger write from pod output.
- No bidirectional memory sync.
- No unknown third-party plugin execution.

## Allowed v1.0 Work

- Plan and validate context packages.
- Validate transfer path safety.
- Validate structured pod return files and metadata.
- Produce proposed updates for human review.
- Record blockers and safety findings.

Remote execution orchestration remains a future research item, not a v1.0
release promise.
