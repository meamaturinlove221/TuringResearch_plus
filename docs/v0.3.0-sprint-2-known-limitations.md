# v0.3 Sprint 2 Known Limitations

Status: integration-gate limitations.

## Web / Apify

- Default mode is fake or local fixture mode.
- Live web and Apify paths are optional and skipped by default in tests.
- `APIFY_TOKEN` is optional and not required for default tests.
- Retrieved web content is not human-verified.
- The adapter does not bypass login, paywall, restricted content, or private
  authorization boundaries.

## Related Work

- Related-work positioning is conservative review material.
- It does not generate final paper paragraphs.
- It does not prove novelty or definitive no-collision.
- Fake method cards, fake citation graph, and fixture pages require manual paper
  review before final use.

## Skill Routing

- Routing is recommendation-only.
- It does not execute skills, spawn agents, or call tools.
- It may return multiple ranked skills when task wording overlaps.

## Vault Graph and Ontology

- Vault graph is a lightweight local graph, not a graph database.
- Ontology SOP output is review material, not final ontology truth.
- Wikilinks are navigation aids, not evidence.
- Evidence-bearing edges require source refs before final claims.

## VGGT Knowledge Pack

- The knowledge pack does not run VGGT or Modal.
- It does not read private VGGT paths.
- It does not package raw data or SMPL-X model files.
- It does not claim SparseConv3D success.
- It does not generate final paper conclusions.
