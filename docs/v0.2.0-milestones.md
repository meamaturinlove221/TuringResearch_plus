# TuringResearch Plus v0.2.0 Milestones

This milestone plan sequences the v0.2.0 backlog without starting implementation.

## Milestone 0: Planning Freeze

- Scope: finalize backlog, milestones, risk register, and priority board.
- Exit criteria:
  - `docs/v0.2.0-backlog.md` exists.
  - `docs/v0.2.0-milestones.md` exists.
  - `docs/v0.2.0-risk-register.md` exists.
  - `lanes/13_v0.2_planning.md` records the planning state.

## Milestone 1: Adapter Safety Foundation

- Backlog items:
  - BL-04 API Key Handling And Live Test Policy.
  - BL-01 Live Semantic Scholar Adapter.
  - BL-02 Live arXiv Adapter.
  - BL-03 Optional Apify Web Adapter.
- Exit criteria:
  - All live adapters are optional.
  - Missing keys produce typed errors.
  - Live tests are skipped by default.
  - Fake-mode parity is preserved.

## Milestone 2: PDF Markdown Phase B

- Backlog items:
  - BL-05 PDF Figure Extraction.
  - BL-06 PDF Table Extraction.
  - BL-07 PDF Section Tree And Page Map Upgrade.
  - BL-08 PDF Quality Report Phase B.
- Exit criteria:
  - Figure/table outputs include provenance.
  - Section tree and page map remain stable.
  - Quality report surfaces warnings rather than overclaiming extraction quality.

## Milestone 3: Graph And Survey Upgrade

- Backlog items:
  - BL-09 Real Citation Graph Optional Expansion.
  - BL-10 Literature Survey Screening Upgrade.
  - BL-11 Evidence Matrix Upgrade.
  - BL-12 Gap Extraction Improvement.
- Exit criteria:
  - Citation graph expansion respects depth and budget.
  - Survey screening records inclusion/exclusion reasons.
  - Evidence matrix and gap outputs require EvidenceRef.

## Milestone 4: Vault Upgrade

- Backlog items:
  - BL-13 Vault Search Upgrade.
  - BL-14 Vault Graph Traversal Improvements.
  - BL-15 Vault Artifact Ingestion Improvements.
- Exit criteria:
  - Search is stronger but local by default.
  - Traversal supports hop limits, edge filters, and evidence paths.
  - Artifact ingestion is idempotent and evidence-aware.

## Milestone 5: Race Mode Upgrade

- Backlog items:
  - BL-16 Upstream Watch From Public Snapshots.
  - BL-17 IdeaCard Scoring Upgrade.
  - BL-18 Feature Capsule Implementation Workflow.
- Exit criteria:
  - Public snapshots are watchable.
  - Unknown sources remain watch-only.
  - Feature capsule workflow remains Source Hygiene gated.

## Milestone 6: Paper Pipeline Upgrade

- Backlog items:
  - BL-19 Better Figure Captions.
  - BL-20 Paper Section Status Upgrade.
  - BL-21 Method Figure Linkage.
- Exit criteria:
  - Captions link to figure provenance.
  - Section readiness reports missing evidence and missing figures.
  - Paper draft gate still blocks missing ExperimentReport.

## Milestone 7: Examples And Release Candidate

- Backlog items:
  - BL-22 v0.2.0 Examples Upgrade.
- Exit criteria:
  - Fake-mode examples pass without API keys.
  - Optional live examples are marked live/manual and skipped by default.
  - `docs/mcp-tools.md`, contracts, release notes, and changelog are synchronized.

## Release Blockers

- Any default test requires network access or real API keys.
- Any live adapter bypasses a service protocol.
- Any output artifact lacks serialization support.
- Any Race Mode workflow promotes a source that did not pass Source Hygiene.
- Any paper draft path fabricates experiment results.
