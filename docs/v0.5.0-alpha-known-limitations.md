# v0.5.0 Alpha Known Limitations

Status: active.

## Productization

- Dashboard is static HTML / Markdown, not a live UI.
- Dashboard is static / lightweight.
- No full SaaS UI is included.
- No login, server, SaaS deployment, or cloud deployment is included.
- Public demo suite is fake/demo only.

## Advisor Export

- Advisor export produces PDF / PPTX plans only.
- No real PPTX/PDF generation by default.
- No real PDF, PPTX, DOCX, or HTML binary export is generated.
- No external office converter is called by default.

## Project Templates

- Project templates are skeletons and require human review.
- Generated templates do not contain observed evidence.
- Templates do not create real contracts beyond README placeholders.

## Packaging

- Package is not published to PyPI.
- No automatic tag or GitHub release is created.
- Many v0.2-v0.5 helpers remain local Python helpers, not frozen public MCP APIs.

## VGGT Replay

- Replay does not run VGGT.
- Replay does not run Modal.
- Replay does not read private VGGT paths.
- SparseConv3D success is not established.
- Missing artifacts remain missing.
- No automatic experiment execution is included.
- No automatic paper conclusion generation is included.

## Safety

- Live adapters remain disabled by default.
- Live adapters are optional.
- Remote sync is optional and safety-gated.
- Default tests do not require API keys.
- No raw data or private model files should be packaged.
