Quickstart

Start here if you want to see TuringResearch without live credentials or private data.

Default Path

The default path is fake/demo and local-first:

  • no API key required;
  • no live network required;
  • no remote execution;
  • no private data;
  • no automatic Evidence Ledger write;
  • human review required before public claims.

Suggested Flow

1. Read the repository README.md. 2. Review docs/quickstart.md. 3. Build or inspect the local docs-site. 4. Open the public demo examples. 5. Use the parity dashboards to understand what is ready and what remains deferred.
