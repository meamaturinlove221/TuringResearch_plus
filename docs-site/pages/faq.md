# FAQ

## Is The Docs Site Deployed?

No. v1.5 prepares docs deployment strategy and build hardening, but deployment
remains manual and out of scope by default.

## Does The Default Path Need API Keys?

No. The public demo and fake/default workflows do not require API keys.

## Does TuringResearch Automatically Run Experiments?

No. Experiment runbooks and ingest expectations are generated for review; they
do not run GPU jobs, remote jobs, or Modal jobs by default.

## Is ARIS Implemented?

No. ARIS remains future reference only and is not in the v1.5 implementation
line.
