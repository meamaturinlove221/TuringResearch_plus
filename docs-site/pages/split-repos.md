# Split Repos

v1.5 prepares manual split execution packs. It does not automatically create
GitHub child repositories or upload data.

## Planned Manual Packs

- main docs and public demo bundle;
- `vggt-case` repo manual pack;
- examples repo manual pack;
- no-deploy release bundle fallback.

## Boundary

- No automatic GitHub repository creation.
- No private data upload.
- No automatic release publishing.
- No PyPI publishing.
- Human review required before any split action.
