# Reference Parity

TuringResearch v1.2 includes review-only parity surfaces for the stable parts of
the tracked reference repos.

## Complete

- Neocortica Session parity
- Neocortica Scholar parity
- Neocortica Web parity
- yogsoth-ai campaign parity
- yogsoth-ai vault / wiki / edge audit parity
- yogsoth-ai ontology SOP parity
- yogsoth-ai convergence / stress test parity
- yogsoth-ai experiment execution parity
- Research Catalog integration

## Deferred

- ARIS study is deferred to v1.3+.
- OS-level plugin sandbox is deferred.
- Default live provider workflows are deferred.

## Rejected

- Unknown remote execution.
- Automatic experiment execution.
- Automatic observed result writes.
- Fake/demo result promotion.
- Paywall bypass.
- Private content fetching.
- Default network access.
