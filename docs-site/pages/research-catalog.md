# Research Catalog

The Research Catalog dashboard shows how campaign routing, skills, vault graph,
stress tests, experiment runbooks, advisor output, and public release gates fit
together.

- no automatic experiment execution;
- no default network;
- no fake/demo result promotion;
- human review required.
