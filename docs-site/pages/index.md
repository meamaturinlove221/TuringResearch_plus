# Overview

TuringResearch is a local-first Research OS for evidence-led research work. It
organizes project context, artifacts, papers, dashboards, release checks, and
public demo material without turning plans or fake outputs into observed
results.

## What It Is

- A review-first repository for research workflow structure.
- A public demo and docs surface for fake/default workflows.
- A parity implementation against stable original reference repo ideas.
- A privacy-first project that keeps live and remote behavior opt-in.
