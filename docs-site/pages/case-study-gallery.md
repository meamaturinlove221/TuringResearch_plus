# Case Study Gallery

The Case Study Gallery is a local-first index of public-safe demo and
case-study entries.

It includes the VGGT public-safe dogfooding case and v1.1 public demo cases for
robotics paper survey, medical imaging experiment planning, software tooling
research, and multimodal model evaluation planning.
