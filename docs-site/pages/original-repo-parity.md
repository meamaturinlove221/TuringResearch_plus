# Original Repo Parity

TuringResearch v1.4 completed original repo production parity for the stable
parts of the reference repos. The public docs present this as structural,
runtime, production, and deferred status rather than as a blanket claim.

## Production Parity Areas

- Neocortica Session production parity.
- Neocortica Scholar production parity.
- Neocortica Web production parity.
- yogsoth-ai research engine production parity.

## Deferred Or Disabled

- ARIS runtime.
- cross-model review.
- proof-checker.
- meta-optimize.
- paper-claim-audit.
- default live network.
- default remote command execution.
- automatic experiment execution.
