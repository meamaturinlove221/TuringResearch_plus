# Public Demo

The public demo is fake/demo only. It is built for external review, interviews,
and release checks without using live credentials, private paths, raw research
data, restricted model files, or unsupported experiment claims.

## Demo Surfaces

- original repo parity demo;
- public dashboards;
- public case study material;
- fake Scholar/Web walkthroughs;
- fake Session runtime replay;
- Research Catalog and stress/convergence examples.

## Safety Boundary

- No live provider is required.
- No private data is required.
- No fake/demo result is treated as observed evidence.
- No release or deployment is performed by the demo.
