# Idea Documents

Use this directory for collaborator-submitted ideas that may later become roadmap items, feature capsules, SOPs, or implementation tasks.

## File Location

```text
community/ideas/<github-username>/<short-title>.md
```

Use the template:

```text
community/ideas/_template.md
```

## Good Idea Documents Include

- problem statement;
- target users;
- why it matters for TuringResearch;
- expected artifacts;
- likely modules affected;
- references and attribution;
- risks and non-goals;
- possible conversion path.
